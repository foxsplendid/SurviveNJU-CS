# 信息素养与资源获取

「信息差是大学里最可怕的差距之一。很多时候，你学得痛苦并不是因为你不够聪明，而是因为你不知道有更好的学习资源和工具存在。」

在计算机领域，高效检索、筛选、整理信息的能力（即信息素养）甚至比单纯的编程技巧更为关键。本指南将帮助你掌握南大校内外的核心资源获取渠道，并推荐一套高效的工具链。

---

## 🔍 第一步：学会搜索引擎的高级技巧

在开始学习任何专业课之前，请确保你已经掌握了如何使用搜索引擎：

*   **避开“内容农场”**：在 Google/Bing 搜索技术问题时，优先阅读 Stack Overflow、官方文档（如 MDN、cppreference）、GitHub Issue、Medium，警惕各种粗制滥造的商业博客。
*   **搜索语法**：
    *   `site:github.com "nju"`：在 GitHub 域名下搜索与南大相关的所有项目。
    *   `filetype:pdf "nju ics"`：寻找南大计算机系统基础课程的公开 PDF 讲义。
*   **用英文搜索**：99% 的编程报错、系统故障，用英文在 Google 搜索都能在 Stack Overflow 上找到前人提问和高赞解答。

## 🏛️ 南大校内核心资源

作为南大在校生，你享有很多专属的优质资源：

### 1. 教务与选课平台
*   **教服平台**：用于选课、查询成绩、申请免听免考、下载培养方案。请时刻关注首页的教务公告。
*   **教学立方**：南大最常用的教学辅助平台。老师会在上面发布课件、布置作业和进行考勤。推荐使用学长开发的工具批量下载课件（见 [实用工具集](../resources/tools.md)）。

### 2. 校园网与校外远程访问 (VPN)
*   **南大 VPN (Pulse Secure / Ivanti)**：在校外（寒暑假、宿舍区）访问校内教服平台、图书馆购买的电子数据库时，必须连接南大 VPN。
    *   官方配置教程请在南大信息化建设管理服务中心官网搜索“VPN”。
*   **校园网认证**：在校内遇到网络连接问题，先访问 `p.nju.edu.cn` 进行登录认证。

### 3. 图书馆与学术资源
*   **电子书与数据库检索**：通过南大图书馆官网，你可以免费使用超星数字图书馆、Web of Science、IEEE Xplore、ACM Digital Library 等顶级学术数据库。这对于大二、大三需要写论文或做科研的同学至关重要。

## 💻 GitHub 与开源社区的探索

GitHub 是泛CS同学最大的宝藏。你不需要从零开始摸索，很多学长学姐已经为你整理好了资源。

### 1. 南大专属核心 Topic
在 GitHub 搜索栏输入以下 Topic，能够直接聚合南京大学相关的全部公共仓库：
*   [nju-cs](https://github.com/topics/nju-cs)
*   [nanjing-university](https://github.com/topics/nanjing-university)

### 2. 经典南大社区组织
*   **nju-lug**：南京大学 Linux 用户协会。维护了南大学位论文模板（NJUThesis）和各类实用开源工具。
*   **NJU-ProjectN**：蒋炎岩老师等维护的系统能力培养课程组，存放了 NEMU、AM 等核心系统课程的实验框架。

## 🌍 外部自学宝藏平台

*   **[csdiy.wiki (CS自学指南)](https://csdiy.wiki)**：由北大 pkuflyingpig 创建的超高质量自学网站，系统整理了国内外顶级高校（MIT, UCB, CMU, Stanford）的经典 CS 课程及其实验（Labs）路线。
*   **[南哪指南 (nju.today)](https://nju.today/)**：南大学生自发维护的校园生活百科，涵盖衣食住行、选课吐槽等。
*   **Bilibili**：是看国内外公开课视频的首选平台，如蒋炎岩老师历年操作系统录像、魏恒峰老师的离散数学等。

## 🛠️ 推荐的高效学习工具链

| 工具名称 | 工具类型 | 推荐理由 | 链接 |
| :--- | :--- | :--- | :--- |
| **Obsidian** / **Notion** | 个人知识库 | 支持 Markdown、双向链接，适合整理极其复杂的CS知识网络 | [Obsidian](https://obsidian.md) |
| **Zotero** | 文献阅读与管理 | 免费开源，一键抓取论文元数据，做科研和毕业设计必备 | [Zotero](https://www.zotero.org) |
| **Anki** | 卡片记忆 | 基于间隔重复算法，适合记忆基础概念、Linux 常用命令、期末闭卷考点 | [Anki](https://apps.ankiweb.net) |
| **Xboard / Clash** | 网络代理工具 | 用于顺畅访问 GitHub、Google Scholar 等开发和科研必备网站 | 自行探索 |

!!! tip "网络代理与学术科研"
    为了在大学里顺利地查阅英文资料、克隆 GitHub 代码、使用学术搜索，建议大一入学就尽早配置好稳定的网络代理。如果你有困惑，可以向周围的学长或加入技术社团进行交流。
