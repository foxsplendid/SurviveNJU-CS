# 信息素养与资源获取

信息素养包括检索、判断来源、核对版本、管理引用和保护账号数据。它与编程、数学和领域知识相互补充，不能用单一能力替代另一项能力。

本页整理南京大学校内服务、公开资源与常用工具入口，并说明核验和安全边界。

---

## 🔍 第一步：学会搜索引擎的高级技巧

在开始学习任何专业课之前，请确保你已经掌握了如何使用搜索引擎：

*   **核验来源层级**：技术问题优先查标准、项目官方文档、维护者 Issue 和原始论文。社区问答与博客可以提供线索，但仍要核对版本、发布日期和适用环境。
*   **搜索语法**：
    *   `site:nju.edu.cn "课程名称"`：优先查找南京大学院系、教师或课程组公开页面。
    *   `site:github.com "nju" "项目名称"`：查找相关开源项目，并继续核验仓库所有者、许可证和更新时间。
*   **尝试英文关键词**：编程报错和系统故障通常更容易用英文错误信息定位，但搜索结果仍需结合软件版本和官方文档验证。

!!! warning "公开可访问不等于获准使用"
    不要通过搜索语法绕过课程平台权限，也不要下载来源不明的课件、录像或试卷。只有发布者明确公开且你能确认授权边界的材料，才适合在指南中推荐或再次传播。

## 🏛️ 南大校内核心资源

以下服务的访问资格、资源范围和使用条件由学校及内容提供方决定：

### 1. 教务与课程平台
*   **网上办事大厅与教务系统**：培养方案查询、选课、成绩和学籍业务的入口可能随系统调整。先查看本科生院当期通知，再从学校网上办事大厅进入指定服务，不要依赖旧截图或旧入口。
*   **教学立方等课程平台**：部分课程会在平台发布通知、课件和作业。只使用平台提供或教师明确许可的下载方式，不要向第三方工具提交统一身份认证密码、Cookie 或访问令牌，也不要批量抓取受限课件。

### 2. 校园网与校外远程访问
*   **学校 VPN**：截至 2026-07-09，学校发布了 aTrust VPN 试用通知，原 EasyConnect 服务同时保留。客户端、入口和适用范围以后续信息化部门通知为准，不要沿用旧客户端教程。
    *   当前通知：[南京大学 VPN 系统试运行通知](https://itsc.nju.edu.cn/cc/9c/c21415a838812/pagem.htm)
*   **CARSI**：部分图书馆订购资源支持通过 CARSI 和统一身份认证校外访问，无需先连接 VPN。可用资源清单会变化，请查图书馆说明。
    *   官方说明：[CARSI 校外访问电子资源](https://lib.nju.edu.cn/info/1012/2764.htm)
*   **校园网认证**：在校内遇到网络连接问题，先访问 `p.nju.edu.cn` 进行登录认证。

### 3. 图书馆与学术资源
*   **电子书与数据库检索**：在学校订购范围和许可期限内，可通过图书馆访问电子书、文摘或论文数据库。具体数据库、访问方式和使用条件会变化；不得使用脚本连续、系统地下载文献，也不得向校外人员共享账号或受限全文。

## 💻 GitHub 与开源社区的探索

GitHub 可用于发现公开项目、课程框架和学生整理资料，但搜索排名、星标数和公开状态都不能证明内容正确、获学校认可或具备再分发授权。

### 1. 南大专属核心 Topic
在 GitHub 搜索以下 Topic，可以发现一部分由维护者自行标注的南京大学相关公共仓库；结果不完整，也不代表学校认可：
*   [nju-cs](https://github.com/topics/nju-cs)
*   [nanjing-university](https://github.com/topics/nanjing-university)

### 2. 南京大学相关公开组织
*   **[nju-lug](https://github.com/nju-lug)**：GitHub 公开组织，包含 NJUThesis 等社区项目。组织身份、项目维护者和当前活动应分别查看公开主页与仓库，不能由 GitHub 名称推断学校背书。
*   **[NJU-ProjectN](https://github.com/NJU-ProjectN)**：包含 NEMU、AbstractMachine 等公开系统项目。具体课程应使用任课教师指定的版本；公开源码不等于可提交的作业实现。

## 🌍 外部自学宝藏平台

*   **[csdiy.wiki (CS自学指南)](https://csdiy.wiki)**：学生社区维护的课程索引，可用于发现公开课程；具体课程内容、许可证和时效性仍需回到原课程页面核验。
*   **[南哪指南 (nju.today)](https://nju.today/)**：第三方学生社区内容，可用作校园生活线索；地点、价格、开放时间和办事规则仍需回到学校或服务单位当前页面核验。
*   **公开视频平台**：只使用教师、课程组或版权方明确公开的课程录像；搬运视频和来源不明的录屏不应作为本指南的推荐来源。
