# 转专业与专业准入指南

对于通过大类招生或跨学院申请进入南京大学的同学来说，转入计算机、软件工程、人工智能、智能科学与技术等相关方向，核心不是“听说某门课够了”，而是读懂当年的**学科、专业准入实施细则**。

本页只整理通用流程和备考思路。所有准入课程、名额、成绩要求、考核形式和录取结果，均以南京大学本科生院及目标院系当年通知为准。

---

## 📌 先分清三个概念

### 1. 大类分流
面向按大类入学的学生，在规定时间内进入具体学科或专业。不同大类的分流规则、志愿填报和成绩排序方式可能不同。

### 2. 跨大类（学院）专业准入
面向想进入其他大类、学院或专业的学生。目标专业通常会设置准入课程、成绩要求、容量限制和考核程序。

### 3. 专业准出
转入只是开始。进入目标专业后，还需要完成该专业培养方案中的准出课程、学分和毕业要求；部分原专业课程可能无法完全抵充。

!!! warning "不要把经验帖当政策"
    网上经验帖可以帮助你理解备考节奏，但不能替代当年官方细则。南京大学本科生院公告栏目会发布分流、跨学院准入、选课等通知；目标院系也可能发布更细的补充要求。

---

## 🧭 官方核验清单

每次准备申请前，按这个顺序核验：

1. **本科生院公告通知**：查“分流”“跨学院准入”“转系转专业”“选课”等关键词。
2. **目标院系通知**：查计算机学院、软件学院、人工智能学院、智能科学与技术学院的本科生公告或教学通知。
3. **当年准入课程表**：确认课程名、课程号、学分、成绩门槛、是否允许替代课程。
4. **选课可行性**：确认跨专业选课是否能选中、是否冲突、是否需要教务员或任课教师协助。
5. **考核与录取规则**：确认是否有笔试、机试、面试、材料审核、容量排序或其他限制。

## 🔑 泛CS方向常见关注点（历史经验）

!!! warning "以下不是当年政策"
    下面只是根据往年经验整理的“常见关注点”，用于帮助你提前规划。请不要把它写进申请材料或口口相传为确定规则。

### 1. 计算机相关方向
*   **课程基础**：程序设计、离散数学、数据结构、计算机系统基础等课程常被视为后续学习的关键基础。
*   **能力要求**：编程能力、基础算法、数学证明和系统理解能力通常比“刷简历项目”更重要。
*   **备考建议**：把准入课程的作业独立完成，补齐 C/C++、数据结构和基本算法，不要依赖往年题背答案。

### 2. 软件工程方向
*   **课程基础**：程序设计、面向对象设计、软件工程与计算、团队项目实践是重点。
*   **能力要求**：代码可读性、协作意识、需求分析和工程化习惯会影响后续课程体验。
*   **备考建议**：准备一个能讲清设计取舍的小项目，比堆砌框架名更有用。

### 3. 人工智能方向
*   **课程基础**：数学分析、高等代数、概率统计、离散数学和程序设计通常是 AI 后续课程的基础。
*   **成绩门槛**：往年经验中曾出现过对准入课程成绩的较高要求，例如“80 分左右”的表述；但具体门槛、课程范围和是否执行，必须以当年人工智能学院或本科生院通知为准。
*   **备考建议**：不要只准备机器学习名词。数学基础、证明能力和编程实现能力更能说明你是否适合该方向。

## 🗓️ 备战路线图

如果你立志转入泛CS方向，大一这一年需要尽早规划：

### 🛠️ 大一上学期：筑基与核验
- [ ] **先查当年准入表**：不要凭上一届经验选课。课程名相似不代表课程号、层次和认定关系相同。
- [ ] **尽早处理选课问题**：跨专业选课可能采用申请、抽签或补选规则，遇到冲突时及时联系书院、院系教务员或任课教师。
- [ ] **打牢编程基础**：C/C++、Java 或 Python 只是工具，真正要补的是调试、抽象、数据结构和复杂度分析。
- [ ] **维持原专业基本学业状态**：即使目标是转出，原专业课程大面积不及格也会增加学业风险和后续补修压力。

### 🚀 大一下学期：补齐基础与准备考核
- [ ] **继续修读关键基础课**：按当年准入表补齐目标专业要求的课程，优先保证课程成绩真实可靠。
- [ ] **关注本科生院和院系通知**：集中申请、材料提交和结果公示时间每年可能不同，以官方通知为准。
- [ ] **针对性备考**：
    *   **机试准备**：练习基础数据结构、排序、递归、搜索、图论入门和简单动态规划。
    *   **面试准备**：准备一份言之有物的个人陈述，讲清为什么要转、已补齐哪些基础、如何处理补修压力。

## 💡 避坑指南

*   **不要“静默观望”**：不要等到申请季才第一次读细则。大一入学后就应了解目标专业培养方案和准入规则。
*   **旁听不等于取得学分**：需要成绩认定的课程，必须按学校选课、补选、免修不免考等正式流程办理。
*   **原专业成绩也重要**：完整成绩单反映学习能力和稳定性；同时，转入后补修压力通常不小。
*   **准备补修成本**：转入新专业后，课程链条可能很紧，部分学分无法转换，极端情况下会影响毕业节奏。
