# 转专业完全指南

对于通过大类招生进入南京大学、但在入学后发现自己真正的志趣在于计算机及相关专业的同学来说，南京大学极具包容性的转专业政策为你提供了充足的“逆袭”机会。南大实行**“转出零门槛，转入有要求”**的准入准出机制，这意味着你不需要原专业学院的批准，只需满足目标专业的“准入课程”和考核要求，即可成功转入。

本指南将为你梳理转入泛CS专业的完整政策、典型考核流程及备战路线图。

---

## 📌 南大转专业政策核心：准入制

### 1. 转出零门槛
原专业院系不得以成绩、绩点等理由阻拦学生申请转专业。只要你想转，教务平台上的申请按钮就会对你开放。

### 2. 集中调整时间节点
学校通常在以下两个时间节点提供集中转专业申请机会：
*   **大一下学期（5月左右）**：这是最主要的调整窗口，名额最多。
*   **大二下学期（5月左右）**：面向大二学生的转专业通道，通常需要补齐较多大一专业课，难度相对较大。

### 3. 什么是“准入课程”？
目标专业（如计算机系、软件学院、人工智能学院）会指定若干门大一核心专业课作为“准入课程”。申请转专业的学生**必须在转专业考核前修读完这些课程并获得学分**（通常要求及格或达到一定分数线），才有资格参加转专业面试或笔试。

## 🔑 各大泛CS专业准入课程与考核（典型方案）

!!! warning "以当年细则为准"
    以下准入课程及考核方式基于往年经验整理，每年可能会根据大类培养方案微调。请务必在大一上学期密切关注教务处发布的《本科生转专业实施细则》。

### 1. 计算机科学与技术系 (CS)
*   **典型准入课程要求**：
    *   **第二学期（大一下）结束准入**：通常要求取得至少 1 门核心准入课程学分，主要为《程序设计基础》或《计算机程序的构造和解释》。此外，需修完第一层次大学数学相应学分。
    *   **第四学期（大二下）结束准入**：因涉及补修和专业进阶，要求修完并取得 5 门左右指定的专业核心准入课（包括《离散数学》《高级程序设计》《数据结构》《计算机组成原理》等）。
*   **考核形式**：
    *   **笔试/机试**：重点考察 C/C++ 编程能力、基础数据结构与算法分析（链表、二叉树、简单图算法）。
    *   **面试**：考察数学基础逻辑、计算机系统基本素养、转专业意向以及抗压沟通能力。

### 2. 软件学院 (SE)
*   **典型准入课程**：
    *   《程序设计基础》（通常为 Java，秋季学期）
    *   《软件工程与计算 I》（春季学期）
*   **考核形式**：
    *   **机试**：重点考察代码编写能力、面向对象编程思想和简单的算法实现。
    *   **面试**：偏重于团队协作意识、大型软件项目实践的兴趣及沟通表达能力。

### 3. 人工智能学院 (AI)
*   **典型准入课程及要求**：
    *   《数学分析》（或数学分析A/B）
    *   《高等代数》（或高等代数A/B）
    *   《离散数学》
    *   《程序设计基础》
    *   ⚠️ **核心硬性门槛**：申请转入人工智能学院通常不仅要求修完并取得上述课程的学分，还要求**相关准入课程的成绩原则上达到 80 分及以上**（含 80 分）。
*   **考核形式**：
    *   **复试考核**：通过初步资格审核后，需参加学院组织的专业准入考核（通常为面试或笔试），重点考察数学逻辑思维、对机器学习/概率统计等 AI 基础理论的理解，以及编程实现的潜力。

## 🗓️ 转专业备战路线图

如果你立志在大学转入CS，大一这一年你需要非常有规划地度过：

### 🛠️ 大一上学期：筑基与选课
- [ ] **主动选修准入课**：在选课阶段，通过旁听、选修或跨大类选课，将目标专业的准入课程（如《高级程序设计》）排入自己的课表。如果实在选不上，可以通过教务处申请补选或听课。
- [ ] **打牢编程基础**：大一上学期把 C++/Java 语言学透。推荐利用 GitHub 上的 [NJUCS-Course-Material](https://github.com/zhangyikaii/NJUCS-Course-Material) 查阅往年作业，提前自学。
- [ ] **保持原专业必修课及格**：虽然转出零门槛，但如果原专业必修课大面积挂科，可能会影响转专业面试的印象分。

### 🚀 大一下学期：冲刺与考核
- [ ] **继续选修第二门准入课**：通常是《数据结构》或《软件工程与计算I》。这门课的难度明显提升，需要投入大量时间刷题。
- [ ] **关注教务处通知（4-5月）**：在教务平台正式提交转专业申请。
- [ ] **针对性备考**：
    *   **机试准备**：在 LeetCode 刷 100-150 道基础题，重点是链表、树、排序、递归。
    *   **面试准备**：准备一封言之有物的个人陈述，梳理自己做过的小项目，复习数学分析、线性代数的基础定理。

## 💡 学长避坑指南与实用建议

*   **不要“静默观望”**：不要等到大一下学期快考试了才去了解政策。大一入学就要把准入课选上。
*   **关于“蹭课”与“免听”**：如果课表冲突选不上准入课，主动联系任课老师，说明自己想转专业的意向，请求旁听并参与平时作业和期末考试（部分老师会给予很大支持）。
*   **原专业成绩也很重要**：面试官通常会看你的完整成绩单。如果你在原专业的课程（如数学、英语）中拿到了高分，会向面试官证明你的学习能力和态度，这非常加分。
*   **调整心态**：转专业是一次重新选择的机会，但也伴随着原专业学分无法转换、转入新专业后需要“补修”大量课程的压力。做好吃苦和可能需要延期毕业（一般不需要，但课程会很紧）的心里准备。
