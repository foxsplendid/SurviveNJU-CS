# 新生入门

本章提供专业信息、规则核验方法和常用服务入口。院系设置、培养方案与专业准入政策可能调整，涉及个人选择时应打开适用于本人年级的正式文件。

刚进入大学时，课程安排、专业方向和办事系统可能同时带来较多信息。本章帮助定位一手来源，并把需要尽早确认的事项列成清单。

无论你通过何种招生或分流路径进入当前专业，都应先核对本人学籍、培养方案和可申请的专业准入规则；本指南不根据入学路径推断资格。

## 📅 新生首周行动清单 (Action Checklist)

以下事项适合在开学初尽早确认，具体顺序按学校通知和个人情况调整：

- [ ] **熟悉校园网络与远程访问**：配置校园网，并按服务类型测试学校 VPN 或图书馆 CARSI；不是所有校外资源都必须通过 VPN。
- [ ] **通读培养方案**：通过网上办事大厅的培养方案查询服务或本科生院当期通知，核对适用于本人年级和专业的毕业要求。
- [ ] **配置开发环境**：在自己的电脑上安装并熟悉基础工具（如 VS Code、Git 客户端），迈出编码第一步。
- [ ] **核验学生组织入口**：从学校、院系或组织公开主页确认当前是否招新及联系方式，不通过来源不明的二维码提交个人信息。
- [ ] **仔细阅读 [免责声明与学术诚信](../preface/disclaimer.md)**：明确本指南的使用边界，筑牢学术诚信底线。

!!! tip "遇到冲突信息时"
    记录页面标题、发布日期、适用年级和发布单位。优先采用适用于本人的正式文件；仍无法判断时，向院系教务或对应发布部门确认，不以群聊转述替代。
