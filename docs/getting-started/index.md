# 新生入门

> 「种一棵树最好的时间是十年前，其次是现在。」 —— 欢迎来到南京大学，开启你的泛CS学习之旅。

刚进入大学，面对全新的环境、截然不同的学习方式，以及令人眼花缭乱的专业方向，感到迷茫是再正常不过的事情。本章节旨在帮助你拨开迷雾，快速融入南大的学习生活。

无论你是通过大类招生入学、直接录入CS相关专业，还是立志通过转专业进入CS殿堂的同学，这里的建议都将为你指明方向。

*   **[正确的学习思维](mindset.md)**：探讨如何从高考思维转变到大学思维，正确对待GPA与实践能力，拒绝无意义的内卷。
*   **[信息素养与资源获取](info-literacy.md)**：掌握获取学术资源、利用开源社区以及高效利用校园网络的必备技能。

## 📅 新生首周行动清单 (Action Checklist)

为了让你能有一个平稳且高效的开局，我们建议你尝试在第一周内完成以下事项：

- [ ] **熟悉校园网络与 VPN**：配置好南大校园网，并测试在校外通过 VPN 访问校内教服平台和图书馆资源。
- [ ] **通读培养方案**：在教服平台下载你所在专业（或意向专业）的培养方案，了解毕业所需的必修和选修学分要求。
- [ ] **配置开发环境**：在自己的电脑上安装并熟悉基础工具（如 VS Code、Git 客户端），迈出编码第一步。
- [ ] **加入技术社团/群聊**：关注如南大 Linux 用户协会（NJU-LUG）等技术社团，融入同辈交流圈。
- [ ] **仔细阅读 [免责声明与学术诚信](../preface/disclaimer.md)**：明确本指南的使用边界，筑牢学术诚信底线。

!!! tip "学长寄语"
    大学不是高三的延伸，它是一场探索自我的旅程。不要被开学初期的繁杂信息和同辈压力（Peer Pressure）所左右，按照自己的节奏，脚踏实地前行。南大泛CS的舞台足够大，容得下每一个热爱代码与技术的灵魂。
