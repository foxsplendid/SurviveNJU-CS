# 南京大学CS相关实验室导航

南京大学在计算机科学、软件工程和人工智能领域有较多活跃研究团队。在本科大二或大三，主动了解并尝试参与实验室训练，是准备科研、保研、出国申请或提升工程能力的常见路径之一。

本篇指南汇总南京大学泛CS相关的代表性实验室和研究组（Research Groups），用于帮助本科生建立检索入口。它不是全量名单；研究团队名称、负责人和方向可能调整，请以各学院官网、教师主页和最新招募通知为准。

## 🔎 各大研究组详解

### 1. LAMDA 研究所 (机器学习与数据挖掘)
*   **地位**：南大 AI 与机器学习方向的重要研究机构，由周志华教授创立，在机器学习理论、多标签学习、弱监督学习和强化学习等方向具有较强影响力。
*   **代表方向**：
    *   **强化学习组**（俞扬 教授）：聚焦于决策优化、仿真环境生成等前沿强化学习研究。
    *   **大模型推理与规划**：研究大模型的泛化能力与多模态对齐。
*   **如何加入**：对本科生的数学功底（线代、概率论、最优化）和编程能力要求极高。大二或大三可关注 LAMDA 每年面向南大本科生开设的暑期培训班或科研实习招募。

### 2. NJUNLP 研究组 (自然语言处理)
*   **地位**：国内历史最悠久的 NLP 研究组之一，在机器翻译（如南大机器翻译系统）等领域底蕴深厚。
*   **代表方向**：机器翻译、大语言模型对齐与微调、自然语言推断。
*   **如何加入**：建议修读完数据结构，掌握 Python 编程与基本的 PyTorch 知识，通过邮件向黄书剑老师等骨干教师申请进组。

### 3. PASCAL 研究组 (静态程序分析)
*   **地位**：由李樾和谭添老师领衔。其开源的 **Tai-e (太阿) 静态分析框架** 是目前国内外静态程序分析教育与学术界的热门系统。
*   **代表方向**：Java 静态程序分析、指针分析、静态缺陷与安全漏洞检测。
*   **如何加入**：修读完《软件分析》或《编译原理》且成绩优秀，代码能力扎实（Java）。

## 💡 本科生进组套磁建议

1.  **准备好学术简历 (Resume)**：
    *   简历里列出你的绩点排名、代表性课程项目或科研训练经历，以及你熟练掌握的编程语言和工具。
2.  **阅读老师的最新论文**：
    *   在发邮件前，至少阅读 1-2 篇老师近期论文或项目介绍（见 [科研入门](../career/research.md) 章节的读论文方法），了解老师最近在做什么。在邮件里具体写写你对该研究的理解，比泛泛群发更有说服力。
3.  **主动展示你的代码能力**：
    *   如果你的 GitHub 上有工整、可运行、有基本测试或文档的课程项目，可以附上链接，帮助老师判断你的工程基础。
