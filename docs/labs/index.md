## 🔎 4. 本科生联系研究组前的核验清单

1. **核对身份**：从学院官网进入教师主页，确认当前单位、职称、研究方向和官方邮箱。
2. **核对活动**：查看近期论文、项目、课程和公告，判断研究组是否仍在该方向开展工作。
3. **核对招募**：只有当前公告明确写出的年级、课程、能力或时间要求才是可引用条件；不要把经验帖中的门槛当成统一规则。
4. **核对任务**：沟通具体任务、指导人、时间投入、数据与代码权限、署名和退出机制。
5. **核对招生**：本科科研参与、开源贡献和导师沟通均不替代院系正式招生考核。

联系邮件和科研准备可参考 [科研入门](../career/research.md)。
