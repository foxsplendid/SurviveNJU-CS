# 境外升学与申请手册

境外硕士和博士项目在学位类型、申请材料、资助方式、导师制度、费用和签证要求上差异很大。录取结果取决于项目当年的完整评审，不能根据单一分数、学校背景或往届案例作保证。

---

## 📐 1. 留学申请的核心要素

先从目标项目官网建立要求清单，再准备材料。下列项目常见但并非每个项目都要求：

```
学业记录 ─────────────> 成绩单、课程要求、学位与先修课条件。
语言与标准化考试 ─────> 托福、雅思、GRE 等是否必需及最低要求。
科研或项目经历 ───────> 与申请方向相关的工作、产出和个人贡献。
推荐信 ───────────────> 数量、推荐人资格、提交方式和截止日期。
申请材料 ─────────────> CV、个人陈述、研究计划、作品集或写作样本。
费用与合规 ───────────> 申请费、学费、资助、存款证明、签证和出口管制等要求。
```

## 🏆 2. 核心开源飞跃手册：nju-cse-flyers

往届案例可以帮助发现项目和准备事项，但样本通常不完整，且录取标准会变化。

* **社区入口**：[nju-cse-flyers GitHub 组织](https://github.com/nju-cse-flyers)
* **适合用途**：查看自愿公开的个人经历、项目名称和材料准备思路。
* **局限**：它不是南京大学或目标高校的官方项目；样本存在自选择和幸存者偏差，也无法反映未公开申请者或最新招生政策。

## 🗓️ 3. 反向规划方法

不要使用统一的“黄金时间线”。先为每个目标项目记录官方截止日期，再为考试送分、成绩单、推荐信、文书、资金和签证预留时间。

| 阶段 | 建议任务 |
| :--- | :--- |
| **探索项目** | 区分授课型硕士、研究型硕士和博士，核对学位、方向、导师制度、费用与资助。 |
| **核对要求** | 保存项目当年官网、申请系统、材料清单、考试政策和截止日期；注意时区。 |
| **准备材料** | 完成语言或标准化考试，联系推荐人，整理 CV、成绩单、个人陈述及项目要求的其他材料。 |
| **提交与跟踪** | 通过官方系统提交，保存付款和提交记录，跟踪补件、面试和结果通知。 |
| **比较结果** | 对比总成本、资助条件、导师与课程、毕业要求、签证限制和退出选择，不只比较学校名称。 |

海外暑期科研是否开放、持续多久、是否有薪、是否提供住宿或推荐信，都取决于具体项目和导师的正式说明，不应预设统一模式。

## 💡 4. 风险控制

1. **不要建立通用权重表**：GPA、研究经历、推荐信和考试成绩的重要性随项目变化。先满足官网写明的硬性要求，再判断如何分配时间。
2. **核验中介与第三方服务**：确认服务范围、收费、退款、数据处理和文书署名；账号、推荐信和真实性声明应由申请人本人控制，不提交虚假材料。
3. **避免把“保底”当保证**：往届录取案例不能给出个人录取概率。申请组合应同时考虑学术匹配、成本、风险承受能力和可接受的备选路径。
4. **单独核对资金与签证**：奖学金名称不等于覆盖全部成本；录取后仍需核对资助期限、续奖条件、保险、税务、签证和入境要求。
