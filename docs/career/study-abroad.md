# 出国留学与飞跃手册

对于南京大学泛CS专业的同学来说，前往海外（北美、欧洲、新加坡、香港等）的高校攻读 Master (硕士) 或 PhD (博士) 学位，是常见的深造路线之一。

南大的院校背景和相关学科基础能提供一定帮助，但录取结果仍取决于 GPA、科研、推荐信、语言成绩、项目匹配度和申请年份竞争强度。

---

## 📐 1. 留学申请的核心要素

海外高校在评估申请者时，主要考察你的**软硬件综合背景**：

```
硬件背景 (硬门槛)：
GPA (学分绩) ─────────> 常见硬指标之一；不同项目对分数和排名的偏好差异很大。
语言成绩 ────────────> 托福、雅思、GRE 等要求以目标项目当年官网为准。

软件背景 (决定上限)：
科研与推荐信 ─────────> 博士申请尤其看重研究经历、推荐信内容和导师/方向匹配。
个人文书 (SOP / CV) ──> 结构化地展示你的学术动机、项目经历与未来研究兴趣。
```

## 🏆 2. 核心开源飞跃手册：nju-cse-flyers

在申请过程中，了解往届相同背景学长学姐的录取情况，是最宝贵的参考数据。

*   **南大CS留学飞跃手册组织：nju-cse-flyers**
    *   **链接**：[nju-cse-flyers 官方 GitHub 组织](https://github.com/nju-cse-flyers)
    *   **项目定位**：由南京大学计算机科学与技术系、软件学院、人工智能学院历年出国留学的同学自发维护的非官方飞跃手册（Flyer Manuals）。
    *   **核心内容**：
        *   **往届生背景与去向库**：收录部分申请者的 GPA 排名、托福/GRE 成绩、科研发表情况以及最终录取去向。它适合做定位参考，但样本有自选择偏差，不能当作录取概率模型。
        *   **申请心路历程**：详细分析了如何选择中介（甚至全DIY）、如何修改 SOP 个人陈述、以及如何联系海外暑期科研。

## 🗓️ 留学申请黄金时间线（以大四秋季提交申请为例）

### 🛠️ 大一至大二：打磨绩点，备考英语
- [ ] **重视 GPA**：GPA 是留学申请的重要硬指标之一，越早稳定越好；目标区间应结合申请方向和项目层次判断。
- [ ] **英语起步**：大二暑假或寒假，集中备考并参加一次托福/雅思考试，测试水平。

### 🚀 大三整年：科研进组与暑研套磁
- [ ] **加入实验室**：大三上尝试加入与你兴趣匹配的研究组进行学术训练，争取形成可展示的项目或论文经历。
- [ ] **备战 GRE**（如需）：大三寒假考出可用的 GRE 成绩。
- [ ] **海外暑研套磁（1月 - 3月）**：向海外高校的教授发送套磁信，申请大三暑假去对方实验室做为期 3 个月的无薪/有薪**暑期科研 (Summer Research)**。这是拿海外教授强力推荐信的黄金机会。

### 🎯 大四上学期（9月 - 12月）：文书与投递
- [ ] **撰写文书**：打磨个人陈述 (SOP) 与学术简历 (CV)。
- [ ] **获取推荐信**：联系南大的导师和海外暑研导师，确认他们愿意提交推荐信。
- [ ] **正式提交申请（11-12月）**：大部分北美/新加坡高校的 PhD 申请在 12 月初截止。

## 💡 学长留学避坑忠告

1.  **“推荐信重于一切 (针对 PhD)”**：
    *   如果你申请的是学术型硕士或博士，一封来自本校或海外暑研教授的强力、细节详实的推荐信，其含金量远超你多考的几分托福。把时间花在进组实干、写好论文上，不要为了托福从 102 卷到 108 而反复刷分。
2.  **警惕商业中介**：
    *   绝大多数中介并不懂计算机领域的具体研究方向（如他们分不清编译器和操作系统的区别），很容易写出空洞乏味的模板化 SOP 文书。建议采取 **全 DIY** 或仅使用中介进行基础流程跟进，文书的学术部分一定要自己亲自把关、修改。
3.  **合理搭配保底学校**：
    *   CS 申请近年来极度内卷。在投递时，合理划分“冲刺-主申-保底”梯度。多参考 `nju-cse-flyers` 中的同分数段去向，避免因为盲目自信而面临全拒德（全拒信）的窘境。
