# 考研指南

如果你在大三评估自己的 GPA 排名后，发现自己不具备保研资格（或者希望通过考研换一所学校/专业），国内统一招考研究生是你的核心出路。南京大学泛CS专业的考研极具竞争性，但只要你掌握了正确的复习策略，避开信息不对称，上岸的概率依然很大。

本篇指南将为你拆解南大 CS/AI/SE 考研的核心政策、科目与复习路线图。

---

## 🗓️ 考研复习黄金时间线

以大三下学期开始复习为例：

### 🛠️ 第一阶段（3月 - 6月）：一轮打基础
- [ ] **数学一/二**：以高数、线代、概率论教材为基础，配合复习全书，做透基础习题。
- [ ] **专业课 (408/842)**：通读 408 的四本教材（数据结构、计算机组成原理、操作系统、计算机网络）。优先攻克数据结构和计组这两个重难点。

### 🚀 第二阶段（7月 - 9月）：二轮高强度刷题（暑假）
- [ ] **数学**：开启题海模式（如做《张宇1000题》或《李永乐660题》），整理错题集。
- [ ] **专业课**：开始做 408 的《单科指南》（如王道考研系列），将每个知识点的细节摸透。
- [ ] **英语**：每天高强度背单词，开始做 2010 年以前的英语真题阅读。

### 🎯 第三阶段（10月 - 12月）：三轮冲刺与模拟
- [ ] **真题模拟**：每周限时模拟一套数学和专业课真题（最近 10 年真题）。
- [ ] **政治备考**：开始复习政治（刷肖秀荣 1000 题），背诵考前预测（肖四肖八）。
- [ ] **英语**：练习作文模板。

## 💡 复试与机试备考指南

南大考研复试的刷人率非常高，且复试成绩占比很重。复试核心由以下两部分组成：

1.  **复试机试**：
    *   考研复试机试通常包含 3-4 道算法编程题（类似 ACM/CSP 赛制）。
    *   **备考策略**：初试结束后（1-2月），立即在 LeetCode 刷题，重点练习动态规划、图论、深搜/广搜。参考 [NJU-Summary-Vocation-Camp-OJ](https://github.com/SaulZhang/NJU-Summary-Vocation-Camp-OJ) 中收录的复试机试原题。
2.  **复试面试**：
    *   考察你的专业英语水平、本科项目经历或毕业设计。
    *   面试老师非常看重你的**真实动手能力**（如问你：“你在大学写的最大的代码是多少行？”、“解释一下你的毕业设计架构”）。初试分数再高，如果面试中暴露出毫无工程动手能力，也有被刷的风险。
