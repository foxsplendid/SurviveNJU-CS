## 🔗 2. 资料使用顺序

1. 先读目标年度的南京大学招生章程、专业目录和报考单位通知。
2. 再读教育部或教育考试机构发布的考试要求和当年准考安排。
3. 最后用社区笔记了解学习方法，并逐项回到官方来源复核。

以下仓库属于非官方经验索引，内容可能过期或存在个人偏差：

* [nju-kaoyan/nju_csai_kaoyan](https://github.com/nju-kaoyan/nju_csai_kaoyan)
* [nullxjx/NJU-CS-Kaoyan](https://github.com/nullxjx/NJU-CS-Kaoyan)

!!! warning "不要把非官方材料当作招生文件"
    不为来源不明的试题、答案、课件或 PDF 提供真实性与授权背书，也不要重新分发无法确认版权或保密状态的文件。教育部 [2026 年全国硕士研究生招生工作管理规定](https://www.moe.gov.cn/srcsite/A15/moe_778/s3261/202509/t20250918_1413836.html) 对招生考试材料的命制和保密管理有明确要求。

## 🗓️ 3. 备考节奏示例

这是一种任务分解，不是固定月份安排。启动时间应根据基础、目标专业和当年考试日期倒排。

| 阶段 | 主要任务 |
| :--- | :--- |
| **核对阶段** | 锁定招生单位、专业代码和学习方式，保存当年官方目录，确认英语、数学和专业课科目。 |
| **基础阶段** | 按当年考试要求梳理数学、英语、政治和 408 的知识结构，完成基础练习并记录薄弱点。 |
| **强化阶段** | 进行分模块练习和限时训练，复盘错误原因，避免只统计题量。 |
| **冲刺阶段** | 按正式考试时长进行模拟，准备报名、确认、准考和考试当天所需材料。 |

## 💡 4. 复试核验清单

复试比例、成绩构成、是否设置机试、考核内容和录取排序由各招生单位按年度发布，不能用“固定题数”或统一刷人率概括。

1. 从南京大学研究生院的当年复试汇总页进入目标单位细则。
2. 核对复试名单、差额比例、成绩公式、考核形式、材料提交和资格审查要求。
3. 如果细则设置编程考核，再按指定语言、环境和范围准备；没有官方说明时，不预设题数或赛制。
4. 保存通知发布日期和页面版本。后续补充或更正通知优先于经验帖。
