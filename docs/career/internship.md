# 实习与求职指南

对于不打算深造（不保研、不考研、不出国）、立志本科毕业直接进入工业界大厂从事研发工作的同学来说，你的大学重心与升学党完全不同。

计算机是一个高度看重**真实动手实践能力与算法基本功**的行业。在南大，直接就业的同学拥有极高起点，但你必须在大二、大三提前布局，通过高质量的实习经历打通校招绿色通道。

本篇指南将为你拆解求职的备战路线图与面试突击秘籍。

---

## 🚀 1. 为什么“暑期转正实习”是第一生命线？

在校招（秋招/春招）中，大厂释放的大部分 Headcount (招聘名额) 已经在春季的**暑期转正实习**中被消耗殆尽。

*   **黄金时间线**：大三下学期（每年 3月 - 5月），大厂会开启大规模的暑期实习招聘。如果你能拿到暑期实习 Offer，在 6-8 月去公司实习并通过转正答辩，你在 8 月份就能稳拿秋招正式 Offer，免去了秋招残酷的大群面与多轮技术笔试。
*   **简历第一要素**：没有大厂实习经历的简历，在秋招的第一轮简历筛选中很容易被直接淘汰。

---

## 🛠️ 2. 面试备考两手抓：算法题与八股文

想要通过大厂（如字节跳动、腾讯、阿里、美团等）的技术面试，你必须在考前打通以下两道关卡：

### ① 算法题：LeetCode 刷题法
*   **分值占比**：每轮技术面试（通常三轮）的最后 15-20 分钟，必定会有一道手写算法题（类似 LeetCode 中等难度）。如果写不出或有 Bug 无法运行，面试直接被一票否决。
*   **备齐路线**：
    *   **大二**：开始在 LeetCode 刷题。先按照**分类专题**刷（如：双指针、链表、二叉树、滑动窗口、DFS/BFS、基础动态规划）。
    *   **大三**：完成《LeetCode 热题 100》或《剑指 Offer》专题。累计刷题量达到 **250 - 300 道**时，你就能应对 90% 的面试笔试题。
    *   **模拟测试**：考前多参加 LeetCode 每周六的双周赛/周赛，适应在有倒计时压力下写出无 Bug 代码。

### ② 专业八股文：四大核心专业课背诵
技术面试会高强度提问计算机基础知识，你需要把在学校学到的必修课知识与工业界实践相结合：

*   **计算机网络**：
    *   TCP 三次握手与四次挥手（状态转移、为什么是三次、为什么是四次）。
    *   TCP 可靠传输、拥塞控制机制（慢开始、拥塞避免、快重传、快恢复）。
    *   HTTP 与 HTTPS 的区别（对称加密与非对称加密、SSL/TLS 握手流程）。
*   **操作系统**：
    *   进程与线程的区别；协程的概念与应用。
    *   线程间的同步机制：互斥锁、自旋锁、读写锁、条件变量。
    *   虚拟内存管理：分页、页面置换算法、分段。
*   **数据库**：
    *   MySQL InnoDB 引擎的 **B+ 树索引原理**（为什么不用红黑树或哈希表？）。
    *   事务的隔离级别（读未提交、读已提交、可重复读、串行化）及各自解决的问题（脏读、幻读）。
*   **语言特性**：
    *   C++：虚函数表（多态实现原理）、智能指针（`shared_ptr` 循环引用与 `weak_ptr`）、内存模型。
    *   Java：JVM 垃圾回收机制 (GC)、类加载机制、集合类底层实现（`HashMap` 扩容机制）。

## 🗓️ 校招投递与备战时间节点

| 时间节点 | 备战内容 | 目标 |
| :--- | :--- | :--- |
| **大二整年** | 刷 LeetCode 专题题，参与中型大作业开发 | 打牢代码基础，积累简历项目。 |
| **大三上学期** | 准备简历，手写 1-2 个高质量开源项目，刷热题 100 | 简历成型，算法题基本通关。 |
| **大三下 (3-5月)** | 参加大厂暑期转正实习招聘（网申、多轮笔面试） | 拿下暑期转正实习 Offer。 |
| **大三暑假 (6-8月)** | 入职公司进行暑期实习，准备转正答辩 | 拿到转正 Offer，提前结束秋招。 |
| **大四上 (8-10月)** | 参加正式秋季校园招聘（如果暑期未转正） | 参与海投和面试，斩获多方意向。 |

!!! tip "写好简历的黄金法则"
    **“Star 法则表达项目”**。在简历中介绍你做过的大作业或项目时，使用以下结构：**S**ituation (项目背景/难点) -> **T**ask (你的分工) -> **A**ction (你使用了什么技术、做了什么优化) -> **R**esult (最终取得了什么成果，如：接口响应时间降低 30%，测试覆盖率达到 85%)。有量化数据的简历极具说服力。
