# 升学与就业篇

> 「种一棵树最好的时间是十年前，其次是现在。」 —— 无论你的目标是攀登学术高峰，还是在大厂乘风破浪，合理的生涯规划都必须尽早开启。

南京大学泛CS专业的毕业生在就业市场和国内外学术界都拥有极强的竞争优势。在本科阶段，你大致会面临以下五种主流的未来路径抉择：

1.  **国内保研**：通过校内推荐免试资格，直接升入南大本校或国内其他顶尖科研机构。
2.  **国内考研**：通过全国统一的研究生入学考试（初试+复试），报考本校或外校硕士/博士。
3.  **出国留学**：申请海外（北美、欧洲、新加坡、香港等）的高校攻读 Master (MS) 或 PhD 学位。
4.  **直接就业/实习**：大三或大四通过暑期实习、校招，直接进入互联网大厂、外企、国企或量化基金从事研发或算法工作。
5.  **学术科研**：在本科阶段就深度参与实验室科研，立志攻读直博学位。

## 📅 本科生涯核心规划时间线

为了让你能做到心中有数，我们为你整理了一份贯穿大一至大四的全局节点表：

```
  大一阶段               大二阶段               大三阶段 (博弈黄金期)   大四阶段 (收割季)
  适应大学节奏 ───────> 绩点分水岭 ──────────> 保研定位 / 考研备考 ───> 保研推免 (9月)
  保持高GPA基础         进组尝试科研 (SRTP)     出国考托福/GRE          考研初试 (12月)
  学好编程语言         参加核心算法竞赛        暑期实习/秋招 (3-9月)   出国Offer (1-4月)
```

*   **大一：打牢基础，探索志趣**。保持高 GPA，学透编程语言，适应转专业政策，多听听各方向讲座。
*   **大二：方向分水岭**。专业课难度剧增，绩点开始分化。有精力的同学可以开始组队参加数学建模、ACM 竞赛或申请 SRTP（大学生创新训练计划）项目。
*   **大三：决策与冲刺的黄金期**。
    *   **保研党**：争取绩点冲入保研线（前 25%），准备夏令营材料，练习机试。
    *   **出国党**：高强度备考 TOEFL/GRE，套磁海外暑研机会。
    *   **考研党**：大三下学期（3月左右）全面开启 408 专业课与数学一的复习。
    *   **就业党**：高强度在 LeetCode 刷题，打磨简历，在大三春季（3-5月）疯狂投递大厂的**暑期转正实习**。
*   **大四：收割与落子**。保研推免系统开放（9月）、秋招收尾、考研初试（12月）、海外 Offer 陆续雨后春笋般降临。
