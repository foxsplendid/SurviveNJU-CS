# 升学与就业篇

南京大学泛 CS 专业的毕业生有多种发展路径。不同路径的政策、申请窗口和录取标准都会按年份、院系、项目或用人单位变化，规划时应把官方通知作为唯一的时间和资格依据。

1. **国内推免**：先取得本科院校推荐资格，再参加招生单位的接收考核，并通过教育部推免服务系统完成正式流程。
2. **国内考研**：参加全国硕士研究生招生考试的初试和招生单位复试；博士招生是另一套招生流程。
3. **境外升学**：按目标项目官网要求申请硕士或博士项目。
4. **实习与就业**：通过日常实习、暑期实习、校园招聘或社会招聘寻找岗位。
5. **科研训练**：通过课程项目、校级项目、毕业设计或研究组项目了解研究工作，并据此判断是否继续攻读研究型学位。

---

## 📅 本科阶段规划框架

下表是任务顺序，不是固定日历。推免、统考、境外申请和招聘的实际日期必须逐项核对当年官方通知。

| 阶段 | 建议完成的工作 |
| :--- | :--- |
| **大一** | 适应课程与考核方式，打牢数学和编程基础，了解各专业培养方案。 |
| **大二** | 继续完成核心课程，尝试课程项目、竞赛或科研训练，记录自己真正感兴趣的方向。 |
| **大三** | 根据最新政策评估推免、统考、境外升学和就业路径；分别准备所需的成绩、材料、考试或项目经历。 |
| **大四** | 按各官方系统和项目通知提交材料、参加考核并确认结果，同时准备备选方案。 |

!!! warning "不要沿用往年时间表"
    往年经验只能帮助建立任务清单，不能证明本年度仍在同一日期开放、采用同一考试科目或使用同一评价办法。
