## 🔗 开源往年试卷库索引

以下是 GitHub 上由南大同学发起并维护的往年题共享仓库：

### 1. 计算机系专属期末真题库：njucs-past-exam
*   **链接**：[NijikaIjichi/njucs-past-exam](https://github.com/NijikaIjichi/njucs-past-exam)
*   **内容特色**：专门针对计算机系专业课期末考试卷的收集计划。收录了包括高级程序设计、数据结构、离散数学、计算机网络等多门核心课的往年试卷回忆版、部分手写答案以及模拟题。

### 2. 泛CS核心课程大作业与资料共享：NJUCS-Course-Material
*   **链接**：[zhangyikaii/NJUCS-Course-Material](https://github.com/zhangyikaii/NJUCS-Course-Material)
*   **内容特色**：虽然该仓库主要以大作业和实验报告为主，但每个课程的文件夹下，通常也附带了往届同学整理的期末考试卷回忆版与复习提纲。

### 3. 多学科综合复习资料：NJU-Review-Materials
*   **链接**：[idealclover/NJU-Review-Materials](https://github.com/idealclover/NJU-Review-Materials)
*   **内容特色**：南大最大的跨学科课程资料共享计划，在计科（CS）、数理科学（高等数学、线性代数）等板块下，有大量期末模拟题和复习脑图。

### 4. 智能科学与技术学院试卷库：Resources
*   **链接**：[NJUIS-Students/Resources](https://github.com/NJUIS-Students/Resources)
*   **内容特色**：苏州校区智能科学与技术学院的官方/学生共享资源库，内含该学院专属的核心课程试卷。

## 💡 科学使用真题的建议

*   **不要过早依赖真题**：如果还没有通读 PPT，没有理解基本定理，一上来就直接刷真题只会让你产生巨大的挫败感。建议在基础概念复习完 70% 之后，再开启真题自测。
*   **注重“订正与反思”**：很多手写答案是往届同学课后自己做的，**不代表官方标准答案**，难免存在错漏。在核对答案时，多与同学讨论，如果发现自己的结果与参考答案不一致，倒回去看 PPT 定理，不要盲信参考答案。
*   **合理贡献**：如果你在参加完期末考试后，还能清晰回忆起某些大题，非常欢迎你通过 Pull Request 的形式将回忆版真题贡献到上述开源仓库中，让开源精神在南大代代相传。
