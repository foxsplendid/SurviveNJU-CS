    4.  **侵权快速处理通道**：若有任何资源的权利人、老师、课程组或仓库原作者对本站的链接索引有异议，请直接在 GitHub 提交 [Issue](https://github.com/foxsplendid/SurviveNJU-CS/issues) 告知我们，我们将在收到通知后尽快处理相关链接。
