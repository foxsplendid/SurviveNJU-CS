## ⚖️ 重要版权与公益性质声明

!!! important "必须阅读的原则"

    2.  **不分发受版权保护的文件**：我们坚决尊重南京大学、任课教师以及各开源作者的知识产权。**本站绝不重新上传或直接分发任何受版权保护的考试卷原题、教师 PPT 课件或商业教材**。我们仅提供指向 GitHub 上由学生自发整理的公开仓库的超链接（Hyperlinks）及简要描述。
    3.  **遵循第三方开源协议**：所收录的第三方 GitHub 仓库，其版权与授权协议归原作者所有（如 MIT, Apache 2.0, CC BY-NC-SA 4.0 等）。请读者在参考和引用其代码或笔记前，务必仔细查阅各仓库的 `LICENSE`，并严格遵守。
    4.  **侵权快速处理通道**：若有任何资源的权利人、老师、课程组或仓库原作者对本站的链接索引有异议，请直接在 GitHub 提交 [Issue](https://github.com/foxsplendid/SurviveNJU-CS/issues) 告知我们，我们将在收到通知后尽快处理相关链接。
