# 推荐书单

「在信息碎片化的时代，阅读系统撰写的经典教科书和技术专著，是建立完整知识体系、摆脱‘调包侠’局限的最好方式。」

本篇指南为你精选了在南大泛CS各方向求学期间，最受教授和学长学姐推崇的经典书单，涵盖系统、算法、AI 以及软件工程领域。

---

## 🖥️ 1. 计算机系统方向经典

| 书籍名称 | 作者 | 对标南大课程 | 推荐理由 |
| :--- | :--- | :--- | :--- |
| **《深入理解计算机系统》 (CSAPP)** | Randal E. Bryant | 《计算机系统基础》 | 🏆 **整个计算机教育界的无冕之王**。从程序员的视角串联起编译、链接、汇编、异常控制流、虚拟内存和网络编程。大一/大二必读。 |
| **《操作系统导论》 (OSTEP)** | Remzi | 《操作系统》 | 免费开源。以“虚拟化、并发、持久化”三个板块切入，行文极其风趣幽默，配有大量引导思考的问答，比国内绝大多数 OS 教材更易懂。 |
| **《计算机网络：自顶向下方法》** | James F. Kurose | 《计算机网络》 | 采用自顶向下的物理顺序，配图极其精美，对复杂网络协议的场景模拟十分生动，适合网络方向入门。 |
| **《现代编译原理：C语言描述》** | Andrew W. Appel | 《编译原理》 | 俗称“虎书”，相较于宏大的“龙书”，本书篇幅克制，实践指导意义极强，适合写编译大作业时参考。 |

## 🧮 2. 算法与数学基础方向

| 书籍名称 | 作者 | 对标南大课程 | 推荐理由 |
| :--- | :--- | :--- | :--- |
| **《算法导论》 (CLRS)** | Thomas H. Cormen | 《数据结构与算法》 | 算法领域的百科全书。数学论证极其严密，适合作为工具书随查随用。不建议大一零基础通读。 |
| **《机器学习》 (西瓜书)** | 周志华 | 《机器学习导论》 | 🏆 **国内机器学习的第一圣经**。由南大人工智能学院院长周志华教授撰写，内容全面，公式严密，AI 学院必读。 |
| **《统计学习方法》** | 李航 | 《机器学习导论》 | 篇幅克制，对 SVM、EM 算法等经典统计机器学习模型的推导极为清晰，数学基础扎实。 |
| **《概率论与数理统计》** | 陈希孺 | 《概率论与数理统计》 | 讲解极具启发性，帮助你从本质上理解为什么需要概率思维，而非单纯地套用公式。 |

## 🏗️ 3. 软件工程与程序设计方向

| 书籍名称 | 作者 | 对标南大课程 | 推荐理由 |
| :--- | :--- | :--- | :--- |
| **《代码大全》 (Code Complete)** | Steve McConnell | 《软件工程》 | 软件构建领域的经典。教你如何写出可读性强、易于维护、没有异味的代码，建立良好的编码习惯。 |
| **《重构：改善既有代码的设计》** | Martin Fowler | 《软件工程与计算》 | 详细讲解如何一步步优化代码结构而不破坏其功能，设计模式在工程实践中的直接体现。 |
| **《人月神话》** | Fred Brooks | 《软件工程》 | 项目管理领域的圣经。告诉你“为什么在已经延期的项目中增加人手只会让项目更延期”等工程真相。 |

## 📖 4. 书籍获取渠道指南

1.  **南京大学图书馆检索**：
    *   上述绝大多数经典专业书的中文译本和原版英文书，在南大图书馆（仙林杜厦图书馆 / 鼓楼图书馆）都有极其丰富的藏书。通过图书馆官网的 OPAC 系统检索索书号即可借阅。
2.  **正版合法电子书**：
    *   部分国外优秀教材（如 OSTEP, D2L）的作者在其官方网站上提供了**完整且合法的免费电子版本（HTML/PDF）**，我们在 [在线课程推荐](online-courses.md) 和 [GitHub 公开资源索引](github-repos.md) 中已提供了相关链接。
3.  **校内二手书流转**：
    *   在每年的毕业季，毕业生会在各大书院或二手交易平台（如闲鱼、校内跳蚤市场）低价处理专业书，这是大一新生淘书的极佳机会。
