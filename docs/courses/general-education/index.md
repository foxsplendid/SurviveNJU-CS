# 通识课选择与核验指南

通识课的课程名、教师、学分、容量和考核方式可能随学期变化。本页不提供“给分宽松”排名，也不根据学科类别推断课程难度，而是提供一套可以复核的选课方法。

!!! info "当前信息入口"
    本页核对至 **2026-07-14**。南京大学本科生院会按学期发布课程表，并在页面中提供通识课、科学之光、新生研讨课、悦读计划、美育课程等公共课程附件。选课阶段仍可能调整，最终信息以教服平台为准。

## 选课前需要确认的事实

先从当学期官方课表或教服平台记录以下字段：

| 字段 | 为什么需要确认 |
| --- | --- |
| 课程号与完整名称 | 避免把同名或改名课程混为一谈 |
| 课程类型与学分 | 判断能否计入自己的培养方案 |
| 面向年级、专业和先修要求 | 确认是否有选课资格 |
| 任课教师与开课校区 | 教师和校区可能随学期变化 |
| 上课时间、容量和抽签规则 | 评估时间冲突与选中概率 |
| 考核构成 | 区分考试、论文、展示、项目和过程评价 |
| 签到、迟交与缺勤规则 | 评估自己能否稳定满足课程要求 |
| 补退选时间 | 必须以当学期本科生院通知为准 |

“论文课一定轻松”“闭卷课一定难拿高分”都不是可靠规则。论文、展示、小组项目和闭卷考试分别要求不同能力，实际投入取决于主题、评分标准、协作方式和个人基础。

## 试听与决策流程

1. **选课前**：查官方课表、课程简介和培养方案，排除资格与时间冲突。
2. **第一堂课后**：记录教师公布的考核比例、阅读量、签到和作业节奏。
3. **补退选截止前**：根据自己的学期负担作决定，并再次核对官方截止时间。
4. **修读过程中**：保留课程通知和大纲，方便之后贡献有日期、有证据的经验。

!!! warning "不要把经验写成承诺"
    “给分大度”“稳定 90+”“不考试”“只需千字论文”等结论，除非能够引用当学期课程大纲，否则只能标为个人、特定学期的体验，不能作为课程固定属性。

## 社区信息怎么使用

社区评价可以帮助发现需要进一步核实的问题，但证据等级低于官方课表和当期课程通知。

* **[社区平台]** [南哪指南](https://nju.today/)：内容由社区贡献，可能需要 JavaScript，课程评价的年份和作者背景需自行判断。
* **[学生维护、历史资料]** [NJU-Review-Materials](https://github.com/idealclover/NJU-Review-Materials)：仓库定位是课程复习资料，不应笼统描述为当前通识课评价数据库。使用其中材料前需核验年份、课程号、版权和准确性。

阅读社区经验时，至少检查：评价对应的学期、教师、课程号、评价者投入和考核方式是否与当前版本一致。单条“水课”或“雷课”评价不能代表课程总体情况。

## 百科式课程条目模板

新增通识课记录时，建议包含：

```markdown
### 课程完整名称

- 学期：2026-2027 学年第一学期
- 课程号：以教服平台为准
- 课程类型 / 学分：附官方来源
- 任课教师 / 校区：附官方来源
- 考核构成：注明来源是课程大纲还是个人记录
- 工作量体验：注明样本身份和时间，不预测分数
- 适合人群：描述所需兴趣、阅读、写作或协作能力
- 最后核验：YYYY-MM-DD
```

## 版权与学术诚信

* 只链接公开且授权清晰的外部资源，不重新分发教师课件、试卷、教材电子版或学生作业。
* 不上传可直接提交的论文、展示稿或项目成品。
* 引用学生经验时删除个人敏感信息，并区分事实、意见和推测。
