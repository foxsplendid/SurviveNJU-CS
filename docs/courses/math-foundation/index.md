# 数理基础课程

> 「数学是科学的皇后，而计算机科学是数学最大的游乐场。」 —— 建立深厚的数理底蕴，决定了你作为计算机人所能达到的上限。

在南京大学泛CS专业的培养体系中，数理基础课程占据了极其庞大的学分权重。特别是在大一和大二阶段，你将面临高强度的数学分析、线性代数、概率论以及离散数学等经典大课的洗礼。

很多同学在刚入学时会产生困惑：“我是来学敲代码的，为什么要花这么多时间学这些复杂的数学证明？”

---

## 🔗 数学对 CS 的关键支撑

其实，计算机科学中几乎所有分支的底层逻辑都直通数学：

```
  数学分支                     CS 应用领域
  线性代数 ───────────────> 机器学习 / 深度学习 (矩阵求导, 特征值分解)
  概率论与数理统计 ─────────> 贝叶斯分类, 随机算法, 信息论, 强化学习
  离散数学 ───────────────> 算法分析, 编译原理 (状态机), 关系数据库
  高等数学 / 数学分析 ──────> 连续优化 (最优化梯度下降), 图形学渲染
```

*   **人工智能与机器学习**：人工智能的内核是连续数学的优化和离散数学的数据结构结合。线性代数中的矩阵相乘是大模型训练的核心操作，概率论是所有统计分类和不确定性决策的算法内核，微积分则是梯度下降求极值的数学工具。
*   **算法与系统设计**：离散数学中的图论直接支撑了网络路由算法、数据结构的图查找；布尔代数是逻辑电路设计的基础；数理逻辑则是形式化方法和程序正确性证明的基石。

## 💡 给学弟学妹的复习忠告

1.  **重视数学的物理意义与几何直观**：
    *   不要只死记硬背积分表和矩阵化简公式。理解线性变换的几何投影意义、理解奇异值分解是在做什么、理解期望和方差度量了数据的什么特征。这些几何直观将在你大二、大三看论文时，帮助你快速看懂那些满屏公式的算法设计。
2.  **真题是提分的核心利器**：
    *   南大数学基础课的期末试题，其出题题型和套路往往高度稳定。考前做透最近三年的期末真题，能极大地消除考前焦虑，并帮你轻松拿到及格线以上的分数。
