# 微积分、数学分析与相关课程

微积分研究连续变化，数学分析进一步强调定义、极限与证明的严格性。南京大学不同专业可能采用《微积分》《高等数学》或《数学分析》等课程名称，不能共享一套固定的学分、教师和考试结论。

## 学科知识框架

常见内容包括：

* **极限与连续**：数列和函数极限、连续性、极限定理及严格定义。
* **一元微分**：导数、中值定理、泰勒公式和函数性质分析。
* **一元积分**：不定积分、定积分、微积分基本定理和应用。
* **级数**：数项级数、函数项级数、幂级数与收敛问题。
* **多元微积分**：偏导数、全微分、方向导数、梯度和极值。
* **多重积分与场论初步**：重积分、曲线积分、曲面积分以及相关积分公式。

课程层次不同，证明要求、内容顺序和覆盖范围也会不同；本列表不是考试范围。

## 学习建议

1. 记录定理的完整条件，并用反例理解省略条件会发生什么。
2. 计算题先画出定义域、积分区域或函数图像，再进行代数操作。
3. 对证明题区分“已知条件、目标结论、可用定理和关键构造”。
4. 根据当期课程的习题和样卷安排训练量，不依赖固定分值传闻。
