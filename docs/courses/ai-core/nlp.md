# 自然语言处理 (NLP)

*   **课程学分**：3 学分
*   **授课学期**：大三春季学期
*   **授课教师**：人工智能学院 / 计算机系 NLP 组教师
*   **修读阶段**：人工智能学院选修课 / 计算机系选修课

---

## 🌟 课程简介与地位

自然语言处理（Natural Language Processing, NLP）致力于解决“如何让计算机理解和生成人类自然语言”的终极命题。

南京大学在 NLP 领域拥有极其深厚的科研积淀，**南大自然语言处理研究组 (NJUNLP)** 在机器翻译、信息检索等方向处于国内顶尖水平。

本课程将带你完整经历 NLP 的演进路线：从早期的分词、词性标注、句法分析，到统计语言模型（N-Gram）、基于向量表示的词向量（Word2Vec、GloVe），到基于序列的 RNN/LSTM，再到当今以 Transformer、BERT、GPT 为核心的大语言模型（LLM）时代。

---

## 📚 核心理论与期末考点

### 1. 核心理论知识点
*   **传统文本处理与语言模型**：
    *   分词算法（正向/逆向最大匹配）。
    *   N-Gram 语言模型、平滑技术（如加一平滑、Kneser-Ney 平滑）。
    *   **隐马尔可夫模型 (HMM)**（高频考点）：HMM 的三个基本问题，**维特比算法 (Viterbi)** 手推解码过程。
*   **词向量 (Word Embeddings)**：
    *   Word2Vec 的两种架构：CBOW 与 Skip-Gram。
    *   负采样（Negative Sampling）与分层 Softmax（Hierarchical Softmax）的计算原理。
*   **基于深度学习的 NLP**：
    *   Seq2Seq 架构与经典的注意力机制（Bahdanau Attention）。
    *   **预训练语言模型**：BERT（Masked LM、Next Sentence Prediction）与 GPT（自回归语言模型）的异同与特点（Encoder-only vs Decoder-only）。
*   **大语言模型基础**：Prompt Engineering、上下文学习（In-context Learning）、指令微调（SFT）与 RLHF 机制。

### 2. 课程大作业与实验
通常要求基于 PyTorch，在真实文本语料上完成以下任务：
*   **命名实体识别 (NER)** 或 **词性标注 (POS)**：手写 BiLSTM-CRF 模型或微调 BERT 模型。
*   **情感分析**：对电影评论或社交媒体文本进行分类。
*   **简易聊天机器人**：微调一个小型的开源 LLM，或者实现一个基于检索增强生成 (RAG) 的问答系统。

## 💡 备考与学习攻略

1.  **熟练手推维特比算法**：
    *   期末大题常考对于一个简单的 HMM 模型（给定状态转移概率和发射概率），在给定观测序列的情况下，手算推导概率最大的状态序列。
    *   在草稿纸上跟着例题，手写出动态规划的 $\delta_t(i)$ 状态转移过程。
2.  **关注大模型时代的最新技术**：
    *   现在的期末考试会引入大语言模型（LLM）的基础考点。理解 BERT 的双向注意力与 GPT 的因果注意力（Masked Multi-Head Attention）的区别。
3.  **积极了解 NJUNLP 研究组**：
    *   如果你未来打算在机器翻译、文本生成、大模型微调等方向做科研，建议在大二下学期或大三上学期，主动关注南大 NLP 研究组（NJUNLP）的招募信息。其官方 GitHub 存放了许多高质量的开源项目，对于大作业的选题非常有参考价值。
