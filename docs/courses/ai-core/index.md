# 人工智能学院核心课程

南京大学人工智能学院成立于 2018 年，由国际人工智能联合会（IJCAI）理事会主席、欧洲科学院外籍院士周志华教授领衔创办。学院致力于构建一整套“数学基础硬核、算法理论前沿、实践应用宽广”的 AI 课程体系。

本板块汇总了人工智能学院大一至大三的核心专业课和攻略指南。

---

## 🧭 人工智能学院课程路线图

南大人工智能学院的培养方案极具特色，其核心思想是**“将数学与机器学习理论紧密交织”**。在进入大二、大三的核心 AI 算法课之前，你必须完成极其硬核的数理基础训练：

```
  大一阶段（数理内功）
  高等数学 / 数学分析 ──> 线性代数与矩阵论 ──> 概率论与数理统计 ──> 最优化方法
                                                                     │
  大二、大三阶段（核心算法）                                            │
  机器学习导论 <───────────────────────────────────────────────────────┘
       │
       ├──> 深度学习
       ├──> 自然语言处理 (NLP)
       ├──> 计算机视觉 (CV)
       └──> 知识表示与处理 / 多智能体系统
```

*   **大一：数理筑基**。除了常规的高等数学和线性代数，你还需要学习专门的《最优化方法》（如凸优化、梯度下降算法原理），这是所有机器学习算法收敛性的数学保证。
*   **大二下：《机器学习导论》**。这是南大 AI 学院的王牌立院之课，以周志华教授的《机器学习》（西瓜书）为主要教材，难度极大，含金量极高。
*   **大三：前沿应用与方向细分**。深度学习、自然语言处理、计算机视觉等，进入细分领域的科研与工程实践。

## 💡 给 AI 学院新生的学习建议

1.  **学好数学，这决定了你的上限**：人工智能的本质是数学。如果你只是想调用 `sklearn` 或 `PyTorch` 的接口去跑模型（成为“调包侠”），你完全不需要来南大。南大的目标是培养能够改进算法、提出新模型的科学家与架构师。线性代数中的 SVD 分解、概率论中的极大似然估计、最优化中的 KKT 条件，必须烂熟于心。
2.  **科研与课程并重**：人工智能是一个高度学术导向和研究导向的学科。大二下或大三上，建议结合个人兴趣，主动联系学院的导师（如 LAMDA 研究所的各个课题组），进组当小工，参与真正的学术科研。
