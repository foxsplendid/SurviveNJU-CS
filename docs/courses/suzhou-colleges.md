# 苏州校区泛 CS 院系课程入口

> 页面级最后核验：2026-07-14。

本页只整理南京大学苏州校区三个泛 CS 相关学院的官方入口和已公开课表，不把学院方向介绍、招生新闻或旧专业介绍当作现行培养方案。南京大学当前[学院部门](https://www.nju.edu.cn/xybm.htm)页面分别列有智能科学与技术学院、智能软件与工程学院、机器人与自动化学院。

!!! warning "课表不是培养方案"
    课表只能证明某院系、年级和学期的开课记录，不能单独证明完整毕业要求、课程先修关系或未来安排。本科生院提示，选课阶段至开课 2-3 周内部分课程可能调整，个人所选课程信息以网上办事大厅实时发布信息为准；培养方案应从适用于本人年级和专业的正式入口另行核验。

## 使用边界

1. 从本人网上办事大厅读取适用的培养方案和实时课表，不用其他年级附件替代。
2. 引用具体课程时同时记录学院、专业或年级、学年学期、课程号、性质、学分和来源。
3. 旧页面若仍使用历史隶属关系或专业名称，只能作为历史材料；当前机构身份以校级学院列表和学院官网为准。
4. 查不到培养方案正文时明确记录“未确认”，不从学院研究方向反推本科课程。
