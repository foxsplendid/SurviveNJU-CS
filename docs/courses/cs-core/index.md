# 计算机系核心课程

南京大学计算机科学与技术系拥有一套极其严谨且系统化强的课程体系。在南大，我们非常强调学生“系统能力”（从底层硬件到模拟器、操作系统、编译器直至应用层软件的自顶向下理解）和“算法理论能力”的培养。

本板块汇总了计科系大一至大三全部核心专业课的攻略和学习指南。

---

## 🔗 课程依赖与培养路线

南大计科的核心专业课可以通过以下两条主线展开，它们环环相扣，前置课程基础的好坏会直接影响后置课程的修读体验：

```
主线一：算法与理论线
高级程序设计 (C++) ──> 数据结构 ──> 算法设计与分析 ──> 数理逻辑 / 计算理论

主线二：计算机系统线
计算机系统基础 (ICS) ──> 操作系统 (OS) ──> 编译原理 ──> 双主线交汇 (大作业/毕业设计)
                   └──> 计算机组成原理 ──> 计算机网络
```

*   **系统主线**：以《计算机系统基础 (ICS)》为起点，你需要在一整年的时间里通过动手写代码去理解“程序如何在计算机上运行”。随后是硬核的《操作系统》和要求编写完整编译器的《编译原理》。
*   **算法主线**：以《高级程序设计》为起点，训练你的基本编程思维；在大一下的《数据结构》中建立扎实的基础，最后在大二下通过《算法设计与分析》完成理论闭环。

## 💡 给学弟学妹的建议

1.  **绝不拖延实验**：系统线课程（ICS, OS, 编译）的实验工作量巨大。通常一个 Lab 会给出 2-3 周的时间，但如果等到最后 3 天才动工，极大概率会面临爆零的窘境。
2.  **善于利用助教（TA）和 Office Hour**：不要害怕提问。南大计算机系的课程助教通常非常专业且热心，遇到百思不得其解的 bug，带上你的 GDB 调试记录去 Office Hour 往往能获得醍醐灌顶的启发。
