# 操作系统：设计与实现（OS）

!!! info "2026 课程与历史 OSLab 不是同一套实验"
    [2026 公开课程说明](https://jyywiki.cn/OS/2026/lect1.md) 明确把公开主线描述为围绕 3 学分设计、仅保留 MiniLab，并移除内核内容。这是该公开课程分支的教学说明，不等同于教务系统对所有院系、年级和教学班的学分登记。选课信息仍以培养方案和教务系统为准。

## 课程关注什么

操作系统课程从程序执行、并发、虚拟化和持久化等角度理解系统软件。学习重点通常包括：

- 程序、进程、线程与状态机模型；
- 并发、同步、互斥、数据竞争和死锁；
- 地址空间、虚拟存储与资源隔离；
- 文件、设备、系统调用及持久化抽象；
- 借助 Linux 接口和系统工具观察真实程序行为。

具体主题、实验接口和考核方式随年份变化；公开讲义是对应年份的一手来源，不应由往届经验反推当前考纲。

## 学习方法

1. **先确认年份和分支**：从当年课程首页进入讲义、实验和提交系统，避免混用 2022/2025 的接口与 2026 要求。
2. **从可观察行为调试**：使用 GDB、`strace`、`/proc`、日志和最小测试检查状态变化，不凭结果猜实现。
3. **显式记录并发不变量**：写清共享状态、锁保护范围、获取顺序和允许的线程交错，再验证数据竞争与死锁风险。
4. **区分概念学习和内核实现**：2026 公开主线移除了内核实验；想补充内核知识时，应使用独立公开教材或实验，并避免与当前课程提交混淆。
