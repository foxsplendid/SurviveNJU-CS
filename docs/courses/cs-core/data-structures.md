# 数据结构与算法

*   **课程学分**：4 学分
*   **授课学期**：大一下学期
*   **授课教师**：计科系教师组
*   **修读阶段**：计算机系必修课 / 转专业准入课

---

## 🌟 课程简介与地位

《数据结构与算法》是计算机专业承上启下的核心课程。在此之前，你只掌握了用 C/C++ 编写简单的控制流和读写变量（《高级程序设计》）；而通过这门课，你将真正学会**如何在内存中高效地组织、存储和处理数据**，并能够评估你的程序在时间和空间上的效率。

这门课不仅是后续《算法设计与分析》、《数据库系统》等高年级课程的前置基础，也是你在求职面试时参加各大互联网公司算法机试（LeetCode 刷题）的最底层知识体系。

---

## 📚 核心内容与实验要求

### 1. 核心理论知识点
*   **基础结构**：线性表（顺序表、链表）、栈与队列。
*   **树形结构**：二叉树的遍历、线索二叉树、赫夫曼树（Huffman Tree）、二叉搜索树（BST）、平衡二叉树（AVL 树）、红黑树（B-Tree 和 B+ Tree，通常在大作业或数据库中深入）。
*   **图结构**：图的存储（邻接矩阵、邻接表）、深度优先遍历 (DFS) 与广度优先遍历 (BFS)、最小生成树（Prim, Kruskal 算法）、最短路径（Dijkstra, Floyd-Warshall 算法）。
*   **查找与排序**：散列表（Hash Table）、插入排序、冒泡排序、选择排序、快速排序、归并排序、堆排序（Heap Sort）等。
*   **算法分析**：渐进时间复杂度（大 O 表示法）与空间复杂度评估。

### 2. 课程大作业与实验
这门课通常要求你用 C++ 语言亲手实现上述几乎所有的数据结构，并完成若干个综合性应用实验（例如：手写 Huffman 编码压缩器、使用图算法实现校园导航系统、或者手写一个小型的文件索引系统）。

## 💡 备考与学习攻略

1.  **手绘算法推演过程**：
    *   期末理论考中，**手推算法步骤**是必考题（占比通常高达 40%）。比如：给你一串无序数列，要求你在草稿纸上一步步写出**快速排序的划分过程**或**堆排序的调整步骤**；或者给你一个带权图，要求你写出 Dijkstra 算法中各节点距离数组（Dist）的更新过程。
    *   复习时，千万不要只盯着 PPT 上的代码看，拿出一张白纸，跟着 PPT 的图示手推一遍过程。
2.  **注重 C++ 指针与内存管理**：
    *   在写链表、树和图的实验时，指针的误用（如野指针、内存泄漏、悬空指针）是导致程序 Segment Fault 的最大元凶。必须掌握如何使用析构函数正确释放节点内存。
3.  **大一刷题习惯培养**：
    *   配合课程进度，在 LeetCode 上同步刷对应的专题题目。大一下学期如果能刷完 LeetCode 经典 150 题中的基础数据结构部分，你的编码水平和期末机试成绩将会一骑绝尘。
