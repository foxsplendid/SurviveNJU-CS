# 课程攻略

> 「纸上得来终觉浅，绝知此事要躬行。」 —— 欢迎来到南京大学泛CS课程攻略大本营。

本章节是 SurviveNJU-CS 的核心板块之一。在这里，我们将南京大学泛CS相关院系（计科、软件、人工智能、智科等）的本科及部分研究生核心课程，按照专业方向和学科基础进行了系统化梳理。

我们不仅提供官方课程大纲的提炼，更汇集了多届学长学姐在修读这些课程时的**真实体验、避坑指南、复习秘籍以及最核心的开源资源链接**。

---

## 🗺️ 课程地图与章节导航

本攻略分为以下几个子板块，你可以根据当前的年级和专业方向直接跳转：

```
docs/courses/
├── cs-core/              # 计算机系核心课程 (ICS, OS, 数据结构, 编译原理等)
├── ai-core/              # 人工智能学院核心 (机器学习导论, 深度学习, NLP等)
├── se-core/              # 软件学院核心 (软件工程与计算系列, 软件分析等)
├── math-foundation/      # 数理基础课程 (高等数学, 线性代数, 离散数学等)
└── general-education/    # 通识课推荐 (科学之光, 优质通识课体验)
```

1.  **[计算机系核心课程](cs-core/index.md)**：重点关注蒋炎岩老师组的《计算机系统基础 (ICS)》和《操作系统 (OS)》等王牌系统课。
2.  **[人工智能学院核心](ai-core/index.md)**：聚焦于周志华教授领衔的机器学习课程体系，探索 AI 的算法世界。
3.  **[软件学院核心](se-core/index.md)**：围绕团队协作开发、系统分析和大型软件构建能力展开的硬核工程课。
4.  **[数理基础课程](math-foundation/index.md)**：大学CS的内功修炼，涵盖作为 AI/ML 基石的线性代数和概率论。
5.  **[通识课推荐](general-education/index.md)**：从同学们的真实体验出发，推荐给分优异、内容有趣的通识选修课。

## 📈 南大CS“硬核度”雷达（部分代表课程）

为了让你对未来的课业负担有直观的心理预期，我们根据课程的**编程量、数学要求、考核难度**，为部分代表性课程制作了直观的分级：

| 课程名称 | 编程工作量 | 数理推导度 | 考核压力 | 学长综合评级 |
| :--- | :---: | :---: | :---: | :---: |
| **计算机系统基础 (ICS)** | 🔴 极高 (PA) | 🟡 中等 | 🔴 极高 | 💀 传奇 hard |
| **操作系统 (OS)** | 🔴 极高 (Lab) | 🟡 中等 | 🔴 极高 | 💀 传奇 hard |
| **机器学习导论** | 🟡 中等 | 🔴 极高 | 🔴 极高 | 🧠 烧脑 hard |
| **离散数学** | ⚪ 极低 | 🔴 极高 | 🔴 极高 | 🧠 烧脑 hard |
| **数据结构与算法** | 🔴 高 | 🟡 中等 | 🟡 中等 | 📈 中规中矩 |
| **软件工程与计算 II** | 🔴 极高 (大作业) | ⚪ 无 | 🟡 中等 | 👥 工程协作难 |

## 🤝 期待你的加入

由于课程大纲和任课老师会发生变动，本攻略需要持续地更新与迭代。如果你在修读完某门课后有强烈的心得想与学弟学妹分享，或者发现我们收录的资源已失效，非常欢迎你按照 **[贡献指南](../preface/contributing.md)** 提交你的 PR，在对应课程的“学长学姐经验”板块留下你的笔迹。
