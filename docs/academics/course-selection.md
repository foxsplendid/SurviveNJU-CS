# 选课策略与技巧

在南京大学，选课是一门技术活。合理的课程搭配不仅能让你的学期时间表更加从容，还能直接影响你的绩点（GPA）走势。泛CS专业的必修课通常课业较重（如 ICS、数据结构等），因此，如何巧妙地利用通识课和选修课来调节学业压力就显得尤为关键。

---

## ⚙️ 南大选课机制详解

南大教服平台每学期会发布具体选课通知。不同类型课程规则不同，例如专业课、跨专业课、通识课、体育课可能分别采用教务员导入、申请抽签、直选补选等方式。以下只写通用思路，具体以当学期通知为准。

### 1. 选课申请阶段
*   **规则**：按当学期通知，在教服平台提交专业课、跨专业课、通识课、体育课等选课申请。部分课程可能只是预申请，结束后统一抽签或审核。
*   **策略**：先保证本专业培养方案要求的必修、指选和核心课程，再考虑通识、公选或跨专业课程。

### 2. 结果确认与补选
*   **规则**：选课结果公布后，及时在课表中确认。未选中的课程通常不具有修读资格，随意旁听或换班不一定能获得成绩。
*   **策略**：如果关键课程未选中，尽快联系开课院系或本院系教务员，确认是否还有补选、容量调整或替代课程方案。

### 3. 补退选与退课
*   **开课两周内**：通常可以进行补退选；在此阶段退选的课程一般不记载在成绩单。
*   **开课第 3-8 周**：通常仍可退课，但退选记录会保留在成绩单，注明“退选”。
*   **更晚阶段**：通常不再允许退选。实际时间节点以当学期本科生院通知为准。

## 🎨 通识课选课指南

南京大学要求本科生修读一定学分的通识课程，分为数理科学、生命科学、人文艺术、社会科学等不同板块。

### 1. 避坑与“排雷”
*   在选通识课前，务必登录“南哪指南（nju.today）”或向技术社团的学长学姐打听该课程的口碑。
*   **避开“杀手课”**：部分通识课虽然名字好听，但期末要求写上万字论文或进行硬核闭卷考试，且给分极低。
*   **科学之光系列课程**：南大特色课程，由各学科顶尖院士、教授开设，主要介绍前沿科学发展。通常给分较好，且能拓宽视野，强烈推荐选修。

### 2. 推荐的高口碑通识课（泛CS学生友好）
*   **《科学之光：人工智能前沿》**：零距离接触AI大咖，了解行业趋势，给分优异。
*   **《艺术与人生》类通识**：陶冶情操，期末通常为小组汇报或简单作业，有助于在高强度的代码学习中放松大脑。

## 🔄 “免修不免考”制度

如果你已经通过自学掌握某门课程内容，可以在课程开课后规定时间内申请**免修不免考**。按学生手册中的通用规则，申请需要任课教师、学院审核并报本科生院备案。

*   **流程**：在教服平台或院系规定流程中提交申请 -> 任课教师评估同意 -> 学院审核 -> 本科生院备案。
*   **考核**：通过后可以不随堂听课，但仍须按课程要求提交作业并参加所有考核。
*   **利弊**：它能为你省下随堂听课时间，但仍要完成课程要求；缺少课堂互动和即时反馈，期末或项目考核压力可能更高。

## 📅 泛CS学期规划与课程搭配示例

以下是一份典型的计算机专业同学的学年课程搭配建议，旨在平衡理论、系统与通识课的比例：

### 大一上学期（重在打牢数理与编程基础）
*   **必修硬核**：《高级程序设计 (C++)》（4学分）、《数学分析 I》或《高等数学 I》（5学分）
*   **日常缓冲**：《英语》（2学分）、《通识课》（2学分）、《体育 I》（1学分）
*   **学长点评**：大一上重点是适应大学节奏，把 C++ 学透，多在电脑上写代码。

### 大一下学期（系统能力的第一次考验）
*   **必修硬核**：《数据结构》（4学分）、《数学分析 II》（5学分）、《离散数学》（4学分）
*   **日常缓冲**：《通识课/科学之光》（2-4学分）、《体育 II》（1学分）
*   **学长点评**：《数据结构》加《离散数学》课业量明显上升，必须合理调配时间，尽早做实验，切忌拖延。

!!! tip "选课黄金法则"
    **“硬软搭配，松紧有度”**。每个学期安排的硬核实验课（如 ICS, OS, 软件工程等）最好不要超过两门。多出来的学分空余可以用偏文科的通识课、体育课或英语课来填充，保证精力不被提前透支。
