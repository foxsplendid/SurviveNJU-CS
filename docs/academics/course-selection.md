# 选课策略与技巧

在南京大学，选课是一门技术活。合理的课程搭配不仅能让你的学期时间表更加从容，还能直接影响你的绩点（GPA）走势。泛CS专业的必修课通常课业较重（如 ICS、数据结构等），因此，如何巧妙地利用通识课和选修课来调节学业压力就显得尤为关键。

---

## ⚙️ 南大选课机制详解

南大教服平台的选课系统主要分为三个阶段：

### 1. 预选（意愿值阶段）
*   **规则**：系统会给你分配一定额度的“意愿值”（通常为 100 分）。你可以将意愿值自由分配给想选的课程。意愿值投得越高，选中概率越大。
*   **策略**：对于极度抢手的热门课，直接投满 100 分或 80+ 分；对于本专业的必修课和限选课，一般系统会默认保证，投极少甚至不投（0分）也能选中。

### 2. 正选
*   **规则**：系统公布预选抽签结果。此时如果有课程仍有余量，可以进行“先到先得”的抢选。

### 3. 补退选阶段（开学前两周）
*   **规则**：这是选课的“黄金纠错期”。在开学后的前两周，你可以去旁听不同的课程。如果不喜欢或者觉得不合适，可以直接在系统上退选，**不会在成绩单上留下任何痕迹**。
*   **第三周至第八周（退选期）**：如果你觉得这门课实在学不下去或绩点要崩，可以在此阶段申请退选。注意：此时退选会在成绩单上留下 `W`（Withdrawn，退选）的标记，虽然不计入绩点，但在申请国外高校时，过多的 `W` 可能会带来负面影响。

!!! tip "选课完成温馨提醒：一键同步课表"
    教服平台的课表在手机端查看多有不便。选课确定后，你可以使用由南大同学开发的开源工具，将课表一键同步至你手机系统日历（如 iPhone、Android），实现上课前弹窗自动提醒：
    *   **iOS/Android 桌面直接导入**：使用 👉 **[NJU-Calendar-Importer-Flutter](https://github.com/121mc/NJU_Calendar_Importer_Flutter)**，极其优雅便捷。
    *   **在线通用 .ics 导出**：使用 👉 **[nju-schedule-ics](https://github.com/SuperKenVery/nju-schedule-ics)**，转换为标准格式日历。

## 🎨 通识课选课指南

南京大学要求本科生修读一定学分的通识课程，分为数理科学、生命科学、人文艺术、社会科学等不同板块。

### 1. 避坑与“排雷”
*   在选通识课前，务必登录“南哪指南（nju.today）”或向技术社团的学长学姐打听该课程的口碑。
*   **避开“杀手课”**：部分通识课虽然名字好听，但期末要求写上万字论文或进行硬核闭卷考试，且给分极低。
*   **科学之光系列课程**：南大特色课程，由各学科顶尖院士、教授开设，主要介绍前沿科学发展。通常给分较好，且能拓宽视野，强烈推荐选修。

### 2. 推荐的高口碑通识课（泛CS学生友好）
*   **《科学之光：人工智能前沿》**：零距离接触AI大咖，了解行业趋势，给分优异。
*   **《艺术与人生》类通识**：陶冶情操，期末通常为小组汇报或简单作业，有助于在高强度的代码学习中放松大脑。

## 🔄 “免修不免考”制度

如果你在入学前就已经通过自学完全掌握了某门课程的内容（例如在大一前就精通了 C++ 与基础数据结构），你可以在**开学前两周**向教务处申请**免修不免考**。

*   **流程**：填写免修申请表 -> 任课教师评估同意 -> 本科生院审批。
*   **考核**：通过后，你无需随堂听课，不用提交平时作业，但必须按要求参加期中/期末考试。期末卷面成绩即为你的最终成绩。
*   **利弊**：它能为你省下大量时间去提前修读高年级硬核课或做科研，但风险在于失去了平时分的缓冲，期末必须考高分。

## 📅 泛CS学期规划与课程搭配示例

以下是一份典型的计算机专业同学的学年课程搭配建议，旨在平衡理论、系统与通识课的比例：

### 大一上学期（重在打牢数理与编程基础）
*   **必修硬核**：《高级程序设计 (C++)》（4学分）、《数学分析 I》或《高等数学 I》（5学分）
*   **日常缓冲**：《英语》（2学分）、《通识课》（2学分）、《体育 I》（1学分）
*   **学长点评**：大一上重点是适应大学节奏，把 C++ 学透，多在电脑上写代码。

### 大一下学期（系统能力的第一次考验）
*   **必修硬核**：《数据结构》（4学分）、《数学分析 II》（5学分）、《离散数学》（4学分）
*   **日常缓冲**：《通识课/科学之光》（2-4学分）、《体育 II》（1学分）
*   **学长点评**：《数据结构》加《离散数学》课业量明显上升，必须合理调配时间，尽早做实验，切忌拖延。

!!! tip "选课黄金法则"
    **“硬软搭配，松紧有度”**。每个学期安排的硬核实验课（如 ICS, OS, 软件工程等）最好不要超过两门。多出来的学分空余可以用偏文科的通识课、体育课或英语课来填充，保证精力不被提前透支。
