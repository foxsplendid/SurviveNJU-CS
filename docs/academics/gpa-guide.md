# GPA 与绩点完全解读

在南京大学，“绩点”（Grade Point Average, GPA）是衡量你学业表现的最核心指标。不论你是计划在国内保研、申请出国深造，还是争取各种校内奖学金，GPA 都是第一道硬性门槛。

本篇指南将为你拆解南大绩点的计算机制，并探讨如何理性看待它。

---

## 🧮 绩点计算机制

南京大学采用**百分制绩点系统**（非 4.0 满分制，而是以百分制为基础的学分加权平均分）。

### 计算公式
$$\text{平均学分绩} = \frac{\sum (\text{课程百分制分数} \times \text{该课程学分})}{\sum \text{总学分}}$$

也就是说，如果你某学期修读了以下三门课：

1.  **高级程序设计**：4学分，考了 90 分
2.  **数学分析 I**：5学分，考了 85 分
3.  **通识课 A**：2学分，考了 95 分

你的学分绩计算为：
$$\text{加权学分绩} = \frac{(90 \times 4) + (85 \times 5) + (95 \times 2)}{4 + 5 + 2} = \frac{360 + 425 + 190}{11} \approx 88.64$$

### 百分制与等级制对应关系
南大很多选修课和通识课采用五级等级制（优秀、良好、中等、及格、不及格）进行登记，它们在计算加权学分绩时，会转换为对应的百分制分数计算：

| 等级 | 对应百分制折算分 | 描述 |
| :---: | :---: | :--- |
| **优秀** | 95 | 极佳 |
| **良好** | 85 | 良好 |
| **中等** | 75 | 一般 |
| **及格** | 65 | 及格 |
| **不及格** | 50 | 不及格（无学分） |

## 🎯 GPA 在不同发展路径中的权重

### 1. 国内保研 (Postgraduate Recommendation)
*   **规则**：保研排名纯粹看**必修课与专业限选课的学分绩排名**。
*   **门槛**：在泛CS学院（计科、软院、人工智能），保研名额通常占毕业总人数的 25% - 30% 左右。如果想稳拿保研资格，你的专业学分绩排名必须进入前 **25%**。如果想保去清北、中科院或南大本校王牌实验室（如 LAMDA），通常需要排名进入前 **5% - 10%**。

### 2. 出国留学 (Study Abroad)
*   **规则**：国外高校（尤其是北美和欧洲名校）在评估时，通常会要求你将百分制绩点换算为 4.0 制。
*   **门槛**：
    *   **GPA 3.8+ (4.0制) / 百分制 90+**：申顶级名校（如 CMU, Stanford, UCB）的强有力保障。
    *   **GPA 3.5+ (4.0制) / 百分制 85+**：申请 Top 30 优秀高校的基准线。
    *   *注：出国申请除了看绩点，更看重你的英语成绩（TOEFL/GRE）和科研推荐信。*

### 3. 实习与求职 (Career)
*   **规则**：工业界大厂在招聘时，一般不太关注 GPA，更看重你的**工程实践能力、算法刷题量（LeetCode）以及实习经历**。然而，一些顶级外企（如微软、谷歌、量化基金）或国企研发岗位，依然会对 GPA 设有基本门槛（如 80+ 分）。

## 🔄 补考、挂科与重修机制

如果某门必修课不幸挂科（百分制低于 60 分），你需要及时采取补救措施：

### 1. 补考 (Make-up Exam)
*   **时间**：通常在下一学期的开学第一周进行。
*   **绩点计算**：补考通过后，无论你考了多少分，该课程的百分制绩点在加权平均中**一律折算为 60 分**计入绩点。

### 2. 重修 (Retake)
*   **规则**：如果你对某门已经通过的课程成绩不满意（例如《数据结构》考了 62 分，拉低了平均绩点），或者补考未通过，你可以申请随低年级重新修读这门课程。
*   **绩点更新**：重修完成后，新取得的成绩会**覆盖**原来的低分成绩并计入平均学分绩（虽然教务系统的完整成绩单上可能会保留重修的历史记录，但保研计算只认最新/最高的成绩）。

## 🧠 学长建议：理性对待绩点博弈

不要让 GPA 变成你的心魔。大一时，很多人会因为一次期中考试只有 80 分而感到天崩地裂。对此，我们的建议是：

1.  **明确目标**：如果你坚决不出国也不打算保研，一毕业就想去大厂做开发，那 80-85 分的绩点加上丰富的项目开发经验绝对比纯粹的 95 分绩点更具杀伤力。
2.  **绩点性价比博弈**：部分基础数理课（如《数学分析》）和核心专业课（如《操作系统》）由于学分很高（4-5学分），对平均学分绩的拉动极其明显。在这类大课上多投入精力，其投资回报率远高于花时间去卷一门 2 学分的公共课。
3.  **大一绩点不代表全部**：如果大一上学期绩点不理想，不要气馁。大二核心专业课（ICS, 数据结构, 编译）的学分占比极重，在这些课程中拿到好成绩，依然有机会实现绩点的“触底反弹”。
