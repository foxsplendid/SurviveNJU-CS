# 学业生存篇

> 「工欲善其事，必先利其器。」 —— 掌握大学学业的游戏规则，是实现自我发展的第一步。

大学的学业评价体系是一个复杂的“多因子模型”。除了期末考试那张试卷，你的选课策略、对 GPA 计算规则的理解、考前高效备考的方式，以及对学术红线的坚守，都共同决定了你的学业走向。

本章节将向你详细拆解南大学业生存的核心技巧，帮助你建立健康的学业节奏。

---

## 📍 章节导航

在这一章里，我们将探讨以下主题：

*   **[选课策略与技巧](course-selection.md)**：如何利用教服平台科学选课，合理搭配必修与选修，避开“雷区”。
*   **[GPA 与绩点完全解读](gpa-guide.md)**：深入剖析南大绩点计算方式，了解绩点在保研、出国、求职中的实际权重与博弈。
*   **[期末备考攻略](exam-prep.md)**：如何在期末考试周高效复习，合理分配复习资源，掌握“复习三天冲刺”的科学方法。
*   **[学术诚信](academic-integrity.md)**：深入理解学术诚信红线，特别是泛CS专业中关于作业代码查重、合理参考开源项目的边界问题。

## 💡 学业自检问答

大一上学期结束前，确保你能回答出以下问题：

1.  **我的毕业学分要求是多少？**（可以在培养方案里查到各模块学分要求）
2.  **我的绩点（GPA）是如何被计算出来的？**（加权平均还是算术平均？）
3.  **如果选错了课，我在什么时间点之前可以退选？**（选课开课前两周）
4.  **CS专业课大作业查重的边界在哪里？**（查阅学术诚信章节，防范误伤）

!!! tip "学长建议"
    大学的成绩单（Transcript）是伴随你一生的印记。合理的规划可以让你用更少的时间取得更好的绩点，从而把时间留给更重要的科研、项目实践或个人兴趣。让我们开始了解这套学业“游戏规则”吧。
