# 学业规则与学习安排

课程成绩、学籍处理、推免和其他评价可能采用不同规则。需要分别阅读适用于本人年级的培养方案、学校制度、院系细则和当期课程通知。

本章整理选课、成绩、补考重修、考试准备和学术诚信的核验入口与通用方法。

## 💡 学业自检问答

建议在选课和考试前确认以下问题：

1.  **我的毕业学分要求是多少？**（可以在培养方案里查到各模块学分要求）
2.  **我的绩点（GPA）是如何被计算出来的？**（加权平均还是算术平均？）
3.  **如果选错了课，我在什么时间点之前可以退选？**（以当学期通知为准；2026 年常规春秋学期与暑期学校的窗口并不相同）
4.  **课程对协作、引用、代码复用和 AI 工具有什么书面要求？**（查阅当期课程规则与学术诚信章节）

!!! tip "规划建议"
    成绩单会记录你的课程修读结果，但不同升学、交换和招聘场景采用的课程范围与评价方式不同。先理解适用于本人的培养和考核规则，再在学习质量、课程负担与个人发展之间作取舍。
