## 三、常见开源许可证说明

如果你在使用本站索引的 GitHub 仓库时，不确定该仓库的内容是否可以引用，可以参考下表快速判断：

| 许可证 | 可以引用/修改？ | 可以商业使用？ | 分发时需要做什么 |
|---|---|---|---|
| **MIT License** | ✅ 可以 | ✅ 可以 | 保留原版权声明和许可证文本 |
| **Apache License 2.0** | ✅ 可以 | ✅ 可以 | 保留原声明、说明修改内容，并注意专利授权条款 |
| **GPL v3** | ✅ 可以修改 | ✅ 可以 | 若分发衍生作品，通常需要按 GPL 公开相应源码 |
| **CC BY 4.0** | ✅ 可以 | ✅ 可以 | 署名，说明是否修改 |
| **CC BY-SA 4.0** | ✅ 可以 | ✅ 可以 | 署名 + 相同协议分发 |
| **CC BY-NC-SA 4.0** | ✅ 可以 | ❌ 不可商用 | 署名 + 非商业 + 相同协议 |
| **CC0 / Public Domain** | ✅ 完全自由 | ✅ 可以 | 无要求（但建议署名） |
| **All Rights Reserved（无声明）** | ⚠️ 谨慎！ | ❌ 原则上不可以 | 联系原作者取得授权 |

!!! warning "无 LICENSE 文件不代表可以自由使用"
    GitHub 上大量仓库没有明确的 LICENSE 文件。在大多数司法管辖区，**无明确授权 = 保留所有权利**。遇到这种情况，请直接联系原作者询问使用权限。

!!! warning "课程仓库的许可证不一定覆盖全部文件"
    课程资料仓库常同时包含代码、笔记、图片、课件、试卷、作业说明等多类文件。仓库根目录存在开源许可证，并不必然意味着其中每一份课件、试卷或教材摘录都可被自由复制、再分发或商用。本站在收录此类仓库时应优先标注为“链接索引 / 授权待核验 / 含课程资料风险”。

## 四、学术诚信声明

!!! danger "严禁学术不端行为"
    本站提供的任何代码、作业参考、课程笔记，**仅供学习理解之用**，**严禁直接复制提交为自己的作业或考试答案**。

南京大学对学术不端行为（包括考试作弊、论文或代码抄袭、伪造数据、未经授权使用他人成果等）有明确的处分规定；处分结果取决于事实、证据、课程规则和学校程序，严重情形可能导致记过及以上处分，极端情形可能导致开除学籍。

**本站对因不当使用本站内容而导致的任何学术处分不承担任何责任。**

正确的使用方式：
- 参考思路，独立实现代码
- 理解概念，用自己的语言总结
- 学习别人的解题过程，而不是复制结果
