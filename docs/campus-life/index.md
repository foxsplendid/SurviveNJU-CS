# 校园生活篇

大学生活不只有课程与成绩。稳定的作息、可信的互助关系、适量的课外活动，以及需要时主动寻求支持，都会影响长期学习状态。

本章整理校园生活相关入口与选择建议。组织状态、活动安排、竞赛规则和校内政策都可能变化；涉及报名、费用、评奖或个人权益时，应回到学校部门、组织维护者或赛事主办方的当前页面核验。

## 💡 生活与学习建议

以下是一般性建议，不是统一的生活标准：

*   为睡眠、运动、用餐和休息保留稳定时间，不把长期熬夜当作学习投入的证明。
*   选择社团或竞赛前，先确认自己希望获得的是交流、项目经验还是专项训练，再评估每周投入。
*   当压力持续影响睡眠、饮食、学习或日常功能时，优先联系可信赖的人或专业支持，不必等到问题进一步恶化。
