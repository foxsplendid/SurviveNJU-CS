# 社团与技术组织

在南京大学，加入一个志同道合的技术社团，是你快速拓宽技术视野、打破信息差、结交项目队友的最有效途径。与单纯的娱乐性社团不同，技术类社团能为你提供丰富的学术与工程资源。

本篇指南为你介绍南大最核心的几个泛CS技术社团与组织。

---

## 🐧 1. 南京大学 Linux 用户协会 (NJU-LUG)

**NJU-LUG (Nanjing University Linux User Group)** 是南大最具极客氛围和学术影响力的学生技术社团之一。

*   **组织性质**：由热爱 Linux、开源软件和自由软件技术的南大同学自发维护。
*   **官方网站**：[lug.nju.edu.cn](https://lug.nju.edu.cn)
*   **官方 GitHub 组织**：[nju-lug](https://github.com/nju-lug)
*   **核心活动与贡献**：
    *   **南京大学开源镜像站**：维护了校内极速访问的开源软件镜像，方便同学们配置各种 Linux 发行版与开发环境源。
    *   **技术沙龙与讲座**：定期举办零基础 Linux 入门、Git 实战、Vim 高效配置以及高级操作系统、网络开发等专题技术讲座。
    *   **学位论文模板维护**：官方/非官方学位论文 LaTeX 模板 **NJUThesis** 的核心维护者。
*   **如何加入**：关注微信公众号“南大LUG”或加入其官方 QQ 群。大一新生可以申请加入核心技术部参与运维与开发。

## 🧮 2. 南京大学 ACM 算法队

*   **组织性质**：代表南京大学参加 ICPC (国际大学生程序设计竞赛) 和 CCPC (中国大学生程序设计竞赛) 的官方集训队。
*   **特色**：汇集了南大最顶尖的算法极客。集训队有专门的训练室、系统的算法培训课以及往届学长学姐倾囊相授的算法秘籍。
*   **如何加入**：大一入学时会举办选拔赛，或者在大一期间通过出色的算法竞赛成绩、OJ（在线评测系统）刷题量申请加入。

## 💡 参加技术社团的实际意义

1.  **结交大作业“大腿”队友**：
    *   在软件工程与计算（SEC II）或各种系统大作业中，你需要组建 3-5 人的团队。在技术社团里结识的同学，通常具有极强的自主开发能力和责任心，与他们组队能省去你大量后期重构和擦屁股的痛苦。
2.  **获取第一手学术与求职内推信息**：
    *   社团的群聊是信息流汇集地。已经上岸清北、出国的学长会在群里分享实验室招募信息；已经在各大厂（字节、微软等）工作的学姐会定期发布**免笔试直通面试的内推码**。
3.  **克服“冒名顶替综合征”**：
    *   通过参与社团的技术交流和项目贡献，你能在轻松的氛围中发现“原来大佬也有写出 Bug 的时候”，从而消除同辈压力，建立起对技术的自信心。
