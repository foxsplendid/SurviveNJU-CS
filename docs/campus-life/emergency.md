# 紧急情况与校园求助

本页用于在紧急情况下快速找到正确入口。它不替代公安、急救人员、医生或心理咨询师的专业判断，也不提供医疗诊断。

!!! danger "先选择正确的求助渠道"
    - **正在发生的人身威胁、暴力、违法犯罪或无法保证现场安全**：拨打 **110**；事件发生在校内时，同时联系所在校区保卫处。
    - **需要立即医疗急救或转运**：拨打 **120**。校医院急诊服务电话可协助校内医疗响应，但不能替代 120。
    - **需要心理支持、情绪疏导或咨询**：联系心理咨询室或心理援助热线。心理热线不能替代 110 或 120；如存在迫在眉睫的自伤、伤人或其他人身危险，应先拨打 110/120。

!!! warning "时效性"
    本页所有具体电话均于 **2026-07-14** 按南京大学官方页面核验。号码和服务安排可能调整，非紧急情况下请先重新打开来源页面复核；紧急情况下优先拨打 110/120。

## 报警或求助时说明什么

尽量清楚说明：

1. 所在校区、楼栋、楼层或附近明显地标；
2. 正在发生什么、是否仍有危险；
3. 需要公安、急救、消防还是校园安保；
4. 现场人数及可回拨电话。

在保证自身安全的前提下留在便于接应的位置，不围观、不传播未经核实的信息，也不要公开当事人的姓名、照片、病情或联系方式。
