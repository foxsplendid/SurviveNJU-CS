# 南京大学泛 CS 学习与校园生存指南

> 非官方、公益、非营利；事实信息按来源核验，经验内容标注适用边界。

欢迎来到 **SurviveNJU-CS**，一份面向南京大学计算机科学与技术、人工智能、软件工程及相关方向同学的非官方指南。

本指南整理官方办事入口、公开课程信息、同学经验、学习路线和第三方资源入口，目标是降低信息检索与交叉核验成本，而不是替代官方文件、专业判断或个人决策。

!!! danger "遇到紧急情况"
    人身威胁、暴力或正在发生的违法犯罪请拨打 **110**；需要立即医疗急救或转运请拨打 **120**。校内事件可同时联系对应校区保卫处。查看 [紧急情况与校园求助](campus-life/emergency.md)。
