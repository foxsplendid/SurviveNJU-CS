# 南京大学泛 CS 学习指南

> 非官方、公益、非营利；以学生经验共建和公开资源索引为主。

欢迎来到 **SurviveNJU-CS**，一份面向南京大学计算机科学与技术、人工智能、软件工程及相关方向同学的学习指南。

本指南整理公开课程信息、同学经验、学习路线和第三方资源入口，目标是降低信息检索成本，而不是替代官方培养方案、课程通知或个人判断。
