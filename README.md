## 📖 关于本项目

本项目受 [上海交通大学生存手册（SurviveSJTU）](https://github.com/SurviveSJTU/SurviveSJTUManual) 启发，旨在为南京大学计算机科学与技术、人工智能、软件工程及相关专业的同学提供一份全面的学习指南。

**本项目是一个公益性质的资源聚合平台**，仅为更好地服务南京大学在校同学。我们不重新分发任何受版权保护的内容，而是以链接索引的方式整合 GitHub 上已有的优质资源，并提供原创的经验指南框架。

## 🌟 内容概览

| 板块 | 内容 |
|------|------|
| **新生入门** | 泛CS专业概览、转专业指南、学习思维、信息素养 |
| **学业生存** | 选课策略、GPA解读、期末备考、学术诚信 |
| **课程攻略** | 计算机系、人工智能学院、软件学院核心课程攻略，数理基础，通识课 |
| **资源导航** | GitHub仓库大全、在线课程、实用工具、推荐书单、往年考试卷、论文模板 |
| **升学与就业** | 考研、保研、出国留学（飞跃手册）、实习求职、科研入门 |
| **实验室导航** | LAMDA、NJUNLP、PasaLab 等研究组介绍 |
| **校园生活** | 社团组织、竞赛指南、心理健康 |

## 🤝 如何贡献

我们欢迎所有南京大学在读及已毕业的同学贡献内容！

### 贡献方式

1. **提交 Pull Request**：Fork 本仓库，修改或新增内容后提交 PR
2. **提交 Issue**：对内容有疑问、建议或发现错误，请在 [Issue 区](https://github.com/your-username/SurviveNJU-CS/issues) 提出
3. **投稿经验分享**：如果你希望分享课程攻略、转专业经验、升学/求职经验等，请参考 [贡献指南](docs/preface/contributing.md)

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/your-username/SurviveNJU-CS.git
cd SurviveNJU-CS

# 安装依赖
pip install mkdocs-material

# 本地预览
mkdocs serve

# 访问 http://127.0.0.1:8000
```
